### 核心概念

**点钟（指定技师）**
- 用户主动选择技师
- 技师提成20%（高于排钟）
- 适合：回头客、会员

**排钟/轮钟（系统分配）**
- 系统自动分配空闲技师
- 技师提成15%（基础提成）
- 适合：首次到店、散客
- 价格可能优惠（如立减10元）

**DispatchServiceImpl.java**

```java
package com.zbkj.service.service.impl;

import com.zbkj.service.service.DispatchService;
import org.springframework.stereotype.Service;

@Service
public class DispatchServiceImpl implements DispatchService {
    
    @Autowired
    private TechnicianScheduleMapper scheduleMapper;
    
    @Autowired
    private TechnicianMapper technicianMapper;
    
    /**
     * 排钟：自动分配技师
     */
    @Override
    public Technician autoAssignTechnician(
        Integer storeId,
        String date,
        String timeSlot,
        Integer serviceSkuId
    ) {
        // 1. 查询该时段所有空闲技师
        List<TechnicianSchedule> schedules = scheduleMapper.selectList(
            new LambdaQueryWrapper<TechnicianSchedule>()
                .eq(TechnicianSchedule::getStoreId, storeId)
                .eq(TechnicianSchedule::getWorkDate, date)
                .eq(TechnicianSchedule::getServiceSkuId, serviceSkuId)
        );
        
        List<Technician> availableTechs = new ArrayList<>();
        
        for (TechnicianSchedule schedule : schedules) {
            // 解析时间槽JSON，检查该时段是否可用
            String timeSlotsJson = schedule.getTimeSlots();
            JSONArray slots = JSON.parseArray(timeSlotsJson);
            
            for (int i = 0; i < slots.size(); i++) {
                JSONObject slot = slots.getJSONObject(i);
                if (slot.getString("startTime").equals(timeSlot) 
                    && "available".equals(slot.getString("status"))) {
                    
                    Technician tech = technicianMapper.selectById(schedule.getTechnicianId());
                    availableTechs.add(tech);
                    break;
                }
            }
        }
        
        if (availableTechs.isEmpty()) {
            throw new RuntimeException("该时段暂无空闲技师");
        }
        
        // 2. 排钟规则：优先空闲时间最长的技师
        availableTechs.sort((t1, t2) -> {
            // 查询今日已接单数
            int count1 = getOrderCountToday(t1.getId());
            int count2 = getOrderCountToday(t2.getId());
            
            // 接单少的优先
            if (count1 != count2) {
                return count1 - count2;
            }
            
            // 同等条件下，新人优先（订单总数少）
            return t1.getOrderCount() - t2.getOrderCount();
        });
        
        return availableTechs.get(0);
    }
    
    /**
     * 获取排钟优惠价格
     */
    @Override
    public BigDecimal getAutoDispatchPrice(BigDecimal originalPrice) {
        // 排钟立减10元
        return originalPrice.subtract(new BigDecimal("10"));
    }
    
    private int getOrderCountToday(Integer technicianId) {
        // 查询技师今日已接单数
        String today = LocalDate.now().toString();
        return orderMapper.selectCount(
            new LambdaQueryWrapper<StoreOrder>()
                .eq(StoreOrder::getTechnicianId, technicianId)
                .like(StoreOrder::getReserveDate, today)
        );
    }
}
```

### Step 3: 预约API改造

**BookingController.java**

```java
@RestController
@RequestMapping("/api/front/booking")
public class BookingController {
    
    @Autowired
    private DispatchService dispatchService;
    
    /**
     * 创建预约订单（支持点钟/排钟）
     */
    @PostMapping("/create")
    public CommonResult<String> createBooking(@RequestBody BookingRequest request) {
        
        Integer technicianId;
        Integer dispatchMode;
        BigDecimal actualPrice;
        
        // 判断派单模式
        if (request.getTechnicianId() != null) {
            // 点钟模式
            dispatchMode = 1;
            technicianId = request.getTechnicianId();
            actualPrice = request.getOriginalPrice();
            
            // 检查技师是否可用
            Boolean available = dispatchService.checkTechnicianAvailable(
                technicianId, 
                request.getReserveDate(), 
                request.getReserveTime()
            );
            
            if (!available) {
                return CommonResult.failed("该技师该时段不可用");
            }
            
        } else {
            // 排钟模式
            dispatchMode = 2;
            
            // 自动分配技师
            Technician tech = dispatchService.autoAssignTechnician(
                request.getStoreId(),
                request.getReserveDate(),
                request.getReserveTime(),
                request.getServiceSkuId()
            );
            
            technicianId = tech.getId();
            
            // 排钟优惠价
            actualPrice = dispatchService.getAutoDispatchPrice(request.getOriginalPrice());
        }
        
        // 创建订单
        StoreOrder order = new StoreOrder();
        order.setOrderType(2); // 服务订单
        order.setDispatchMode(dispatchMode);
        order.setTechnicianId(technicianId);
        order.setReserveDate(request.getReserveDate());
        order.setReserveTimeSlot(request.getReserveTime());
        order.setPayPrice(actualPrice);
        // ... 其他字段
        
        String orderNo = storeOrderService.create(order);
        
        return CommonResult.success(orderNo);
    }
}
```

## 📅 Day 4-5: 加钟场景实现

### 核心流程

**场景**：客人正在做60分钟足疗，第40分钟要求加钟30分钟

```
14:00 开始服务（主订单）
  ↓
14:40 客人要求加钟30分钟
  ↓
技师端发起加钟请求
  ↓
系统检查：
  - 技师15:00-15:30是否有空？
  - 房间15:00-15:30是否有空？
  ↓
创建加钟订单（子订单）
  - parent_order_id = 主订单ID
  - is_addon = 1
  - addon_type = 1（延长时长）
  ↓
推送给用户支付
  ↓
用户支付成功
  ↓
自动调整后续排期
  - 如果15:00有后续预约，顺延到15:30
  ↓
15:30 服务完成
  ↓
同时核销主订单+加钟订单
```

### Step 1: 加钟API

**AddonController.java**

```java
@RestController
@RequestMapping("/api/technician/addon")
public class AddonController {
    
    /**
     * 发起加钟请求
     */
    @PostMapping("/request")
    public CommonResult<String> requestAddon(@RequestBody AddonRequest request) {
        
        // 1. 查询主订单
        StoreOrder mainOrder = orderMapper.selectById(request.getMainOrderId());
        if (mainOrder == null) {
            return CommonResult.failed("主订单不存在");
        }
        
        // 2. 检查资源可用性
        String extendStartTime = mainOrder.getReserveTimeSlot(); // 原结束时间
        String extendEndTime = calculateEndTime(extendStartTime, request.getExtendMinutes());
        
        // 检查技师是否有空
        Boolean techAvailable = checkTechnicianAvailable(
            mainOrder.getTechnicianId(),
            mainOrder.getReserveDate(),
            extendStartTime,
            extendEndTime
        );
        
        if (!techAvailable) {
            return CommonResult.failed("技师该时段有其他预约，无法加钟");
        }
        
        // 3. 创建加钟订单
        StoreOrder addonOrder = new StoreOrder();
        addonOrder.setParentOrderId(mainOrder.getId());
        addonOrder.setIsAddon(1);
        addonOrder.setAddonType(1); // 延长时长
        addonOrder.setOrderType(2);
        addonOrder.setUid(mainOrder.getUid());
        addonOrder.setTechnicianId(mainOrder.getTechnicianId());
        addonOrder.setStoreId(mainOrder.getStoreId());
        addonOrder.setReserveDate(mainOrder.getReserveDate());
        addonOrder.setReserveTimeSlot(extendStartTime);
        addonOrder.setServiceDuration(request.getExtendMinutes());
        
        // 计算加钟价格（按分钟比例）
        BigDecimal addonPrice = calculateAddonPrice(
            mainOrder.getPayPrice(),
            mainOrder.getServiceDuration(),
            request.getExtendMinutes()
        );
        addonOrder.setPayPrice(addonPrice);
        
        String orderNo = storeOrderService.create(addonOrder);
        
        // 4. 推送给用户
        pushToUser(mainOrder.getUid(), 
            "您的服务已延长" + request.getExtendMinutes() + "分钟，需补差价¥" + addonPrice);
        
        return CommonResult.success(orderNo);
    }
    
    /**
     * 计算加钟价格
     */
    private BigDecimal calculateAddonPrice(
        BigDecimal originalPrice,
        Integer originalDuration,
        Integer extendMinutes
    ) {
        // 按分钟比例计算
        BigDecimal pricePerMinute = originalPrice.divide(
            new BigDecimal(originalDuration), 
            2, 
            RoundingMode.HALF_UP
        );
        
        return pricePerMinute.multiply(new BigDecimal(extendMinutes));
    }
}
```

## 🐛 常见问题

**Q1: 排钟如何保证公平？**
- 优先分配今日接单少的技师
- 同等条件下，新人优先

**Q2: 加钟时技师有后续预约怎么办？**
- 系统自动顺延后续预约
- 推送通知给后续客人
- 客人可选择接受或改期

**Q3: 加钟订单如何计算提成？**
- 加钟订单提成比例20%（高于基础15%）
- 主订单+加钟订单分别计算提成
