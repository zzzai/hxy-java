# HXY 执行路线图 V2（2026-02-22）

## 1. 近期（1-2周）

1. 清理旧 CRMEB 运行资产，完全切到 RuoYi 主线。
2. 完成支付系统阶段 A 收口与证据归档。
3. 补齐 CI 与上线拦截规则执行入口。
4. 定版“双路径”：
   - 主路径：服务商 + 特约商户 + 分账
   - 过渡路径：总部统收 + 虚拟账户台账（仅直营同主体）
5. 固化门店收银模式：先建订单再生成支付码。

## 2. 中期（2-6周）

1. 服务商真实联调（支付/退款/对账）。
2. 灰度发布与回滚演练。
3. 启动非支付模块二开（商品/会员/门店运营）。
4. 上线特约商户进件自动化流程（申请、状态跟踪、审核回执）。
5. 分账失败重试与分账回退补偿全自动化。
6. 将过渡路径门店逐步迁移至主路径（按门店批次切换）。

## 3. 长期（1-2季度）

1. 构建数据中台与经营分析能力。
2. 完成模块化治理，逐步服务化改造准备。
3. 建立稳定的“1人+AI”交付体系与知识沉淀闭环。
4. 基于门店规模与投诉数据评估“预收款跨店通用”是否启动。
