# HXY 窗口接力手册

## 目标

多窗口并行开发时，保证上下文可持续，不依赖聊天历史。

## 存放规则

1. 路径：`hxy/07_memory_archive/handoffs/YYYY-MM-DD/<topic>.md`
2. 每次窗口切换必须新增或更新对应交接文件。
3. 交接文件必须引用相关代码路径与脚本路径，不能只写结论。

## 必填结构

1. 输入背景
2. 变更清单
3. 风险与回滚
4. 下一窗口接力点

## 最小模板

```md
# <topic>

## 输入背景
- 业务目标：
- 约束条件：

## 变更清单
1. 代码：
2. SQL/配置：
3. 文档：

## 风险与回滚
1. 风险：
2. 监控点：
3. 回滚命令/步骤：

## 下一窗口接力点
1. 先执行：
2. 验证命令：
3. 完成标准：
```
