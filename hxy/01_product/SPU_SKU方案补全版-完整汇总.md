## 💡 关键设计亮点

### 1. 服务类库存模型（重要！）

```
实物商品：下单 → 锁定库存 → 支付 → 扣减库存
服务商品：预约 → 立即锁定时间槽（无论是否支付）
```

**原因**：服务类商品是"技师时间"，预约后技师时间已被占用。

### 2. 价格体系（6层）

```
原价 → 会员价 → VIP价 → 闲时价 → 活动价 → 优惠券
（优先级从低到高）
```

### 3. 数据隔离（4层）

```
代码层（ThinkPHP全局范围）
    ↓
中间件层（请求拦截）
    ↓
数据库层（索引优化）
    ↓
日志层（操作审计）
```
