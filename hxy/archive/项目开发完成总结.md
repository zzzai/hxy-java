### 1. 分布式锁（Redis + Lua）

**实现文件**: `DistributedLockUtil.java`

**核心代码**:
```java
// SET NX EX 原子操作
redisTemplate.opsForValue().setIfAbsent(lockKey, lockValue, timeout, TimeUnit.SECONDS);

// Lua脚本释放锁（防误删）
if redis.call('get', KEYS[1]) == ARGV[1] then
    return redis.call('del', KEYS[1])
end
```

**特性**:
- ✅ 防死锁（超时自动释放）
