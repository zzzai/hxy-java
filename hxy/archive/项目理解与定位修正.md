## 🎯 项目本质

### CRMEB底座

**CRMEB是什么**：
- 开源电商系统（Java版）
- 技术栈：Java + Uni-app + Vue
- 功能：新零售、社交电商、分销、拼团、砍价、秒杀
- 多端支持：微信公众号、小程序、H5、APP、PC

**CRMEB特点**：
- ✅ 代码全开源无加密
- ✅ 独立部署
- ✅ 二开方便
- ✅ 免费商用

### 荷小悦项目

**荷小悦是什么**：
- 足疗预约小程序
- 基于CRMEB二次开发
- 核心业务：预约服务（非商品销售）
- 目标：千店连锁足疗预约平台

**应该这样**：

- **改造订单模块**：商品订单 → 服务预约订单
- **扩展业务模块**：技师管理、排班管理
- **保留电商功能**：会员、优惠券、支付、分销

## 🎯 CRMEB vs 荷小悦

### 业务对比

| 维度 | CRMEB电商 | 荷小悦足疗 | 改造方案 |
|------|----------|-----------|----------|
| **核心对象** | 商品 | 服务项目 | 复用商品表，type=2表示服务 |
| **交易流程** | 下单→支付→发货→收货 | 预约→支付→到店→核销 | 改造订单状态流转 |
| **库存管理** | 商品库存 | 时间槽库存 | 新增时间槽表 |
| **配送方式** | 快递/自提 | 到店核销 | shipping_type=2 |
| **SKU** | 规格（颜色/尺寸） | 时间+技师 | 改造SKU逻辑 |
| **会员体系** | 会员等级/积分 | 会员等级/成长值 | 复用会员表 |
| **营销工具** | 优惠券/拼团/秒杀 | 优惠券/礼品卡 | 复用营销模块 |

### 技术对比

| 模块 | CRMEB | 荷小悦 | 改造策略 |
|------|-------|--------|----------|
| **后端** | ThinkPHP 6 | Java Spring Boot | ✅ 已是Java版 |
| **前端** | Uni-app | Uni-app | ✅ 复用框架 |
| **数据库** | MySQL | MySQL | ✅ 扩展表结构 |
| **支付** | 微信支付 | 微信支付 | ✅ 直接复用 |
| **会员** | 会员系统 | 会员系统 | ✅ 直接复用 |

## 🎯 正确的开发策略

### 策略1：最大化复用CRMEB

**复用模块（80%）**：
- ✅ 用户系统（登录、注册、个人中心）
- ✅ 会员系统（等级、积分、权益）
- ✅ 支付系统（微信支付、余额支付）
- ✅ 优惠券系统（发放、使用、核销）
- ✅ 订单系统（基础流程）
- ✅ 后台管理（用户管理、数据统计）

**改造模块（15%）**：
- 🔄 商品模块 → 服务项目模块
- 🔄 订单流程 → 预约流程
- 🔄 库存管理 → 时间槽管理

**新增模块（5%）**：
- ➕ 技师管理
- ➕ 排班管理
- ➕ 门店管理（扩展）

### 策略2：改造而非重建

**订单表改造**：
```sql
-- 不是新建表，而是扩展eb_store_order
ALTER TABLE `eb_store_order`
ADD COLUMN `order_type` tinyint(1) DEFAULT '1' COMMENT '1=商品 2=服务',
ADD COLUMN `technician_id` int(11) DEFAULT '0' COMMENT '技师ID',
ADD COLUMN `reserve_time` datetime DEFAULT NULL COMMENT '预约时间',
ADD COLUMN `service_duration` int(11) DEFAULT '60' COMMENT '服务时长';
```

**商品表复用**：
```sql
-- 不是新建服务表，而是复用eb_store_product
-- type=1 商品，type=2 服务
UPDATE eb_store_product SET type=2 WHERE id IN (服务项目ID);
```

### 策略3：前端复用CRMEB Uni-app

**CRMEB已有前端**：
- `/root/crmeb-java/crmeb_java/app/` - 已有Uni-app项目
- 包含：首页、分类、购物车、我的
- 包含：商品列表、商品详情、下单、支付

**改造方案**：
- 复用CRMEB的app目录
- 修改首页：商品列表 → 门店列表
- 修改商品详情：商品SKU → 服务预约
- 修改购物车：删除（服务不需要购物车）
- 新增预约页：时间选择、技师选择
