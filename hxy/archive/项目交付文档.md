## 🚀 部署说明

### 环境要求
- JDK 1.8+
- Maven 3.6+
- MySQL 5.7+
- Redis 5.0+

### 快速部署

#### 1. 数据库初始化
```bash
# 方式A：命令行
mysql -u root -p crmeb_java < /root/crmeb-java/hxy/database/database_migration_v1.0.sql

# 方式B：脚本
chmod +x /root/crmeb-java/hxy/tools/scripts/init_database.sh
/root/crmeb-java/hxy/tools/scripts/init_database.sh
```

#### 2. Redis配置
```bash
# 启动Redis
systemctl start redis
systemctl enable redis

# 测试连接
redis-cli ping
```

#### 3. 应用启动
```bash
cd /root/crmeb-java/crmeb_java/crmeb
mvn clean install -DskipTests
mvn spring-boot:run
```

#### 4. 验证部署
```bash
# 健康检查
curl http://localhost:8080/actuator/health

# API文档
http://localhost:8080/swagger-ui.html
```

## 🗄️ 数据库表结构

### 新增表（8个）

1. **eb_technician** - 技师信息表
   - 字段：id, store_id, name, avatar, level, service_years, skill_tags, intro, rating, order_count, status

2. **eb_technician_schedule** - 技师排班表
   - 字段：id, store_id, technician_id, service_sku_id, work_date, time_slots(JSON), total_slots, available_slots, status, is_offpeak_enabled

3. **eb_booking_order** - 预约订单表
   - 字段：id, order_no, user_id, store_id, technician_id, schedule_id, slot_id, service_sku_id, booking_date, booking_time, original_price, paid_amount, is_offpeak, pay_type, member_card_id, status, verify_code

4. **eb_member_card** - 会员卡表
   - 字段：id, card_name, card_type, price, total_times, stored_value, valid_days, allow_partial_use, status

5. **eb_user_member_card** - 用户会员卡表
   - 字段：id, user_id, card_id, card_no, remaining_times, remaining_amount, active_at, expire_at, status, order_id

6. **eb_member_card_usage** - 会员卡使用记录表
   - 字段：id, user_card_id, user_id, order_id, usage_type, used_times, used_amount, before_times, after_times, before_amount, after_amount, store_id, technician_id

7. **eb_stock_flow** - 库存流水表
   - 字段：id, store_id, sku_id, order_id, change_type, change_quantity, before_available, after_available, before_locked, after_locked, operator_id, operator_type, remark

8. **eb_offpeak_rule** - 闲时优惠规则表
   - 字段：id, rule_name, store_id, discount_rate, time_periods(JSON), weekdays, is_enabled, priority

### 扩展表（1个）

9. **eb_store_product_sku** - 扩展库存字段
   - 新增字段：available_stock, locked_stock, sold_stock

## 💡 技术亮点

### 1. 分布式锁（Redis + Lua）
- SET NX EX 原子操作
- Lua脚本释放锁（防误删）
- UUID校验（防止误删他人锁）
- 超时自动释放（防死锁）

### 2. JSON时间槽设计
- 灵活扩展（无需改表结构）
- 动态定价（支持闲时优惠）
- 状态追踪（锁定时间、订单ID）
- 高性能（单次查询获取全部信息）

### 3. 三重并发控制
- Redis分布式锁（防止并发请求）
- MySQL行锁（FOR UPDATE）
- 原子化更新（WHERE条件二次校验）

### 4. 库存流水追踪
- 完整记录变动（入库/锁定/释放/扣减/退款）
- 变动前后库存快照
- 操作人信息
- 关联订单ID
- 支持对账审计

### 5. 会员卡部分核销
- 次卡：一次使用多次
- 储值卡：一次使用部分金额
- 使用流水：完整记录
- 并发安全：分布式锁保护
