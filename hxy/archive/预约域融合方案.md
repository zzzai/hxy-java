# 预约域融合到CRMEB Java项目方案

> **目标**: 将DDD架构的预约域代码融合到CRMEB Java原项目  
> **原则**: 保持CRMEB原有架构，适配其三层结构

---

## 📊 架构对比

### DDD架构 → CRMEB架构映射

| DDD层 | CRMEB层 | 说明 |
|-------|---------|------|
| Domain（领域层） | Service（业务逻辑） | 业务规则放在Service |
| Application（应用层） | Service（应用服务） | 编排多个Service |
| Infrastructure（基础设施） | Dao（数据访问） | MyBatis Plus |
| Interfaces（接口层） | Controller（控制器） | REST API |

---

## 🎯 融合策略

### 策略1：保留核心业务逻辑

**保留**：
- ✅ 业务规则（预约时间校验、状态流转）
- ✅ 核心算法（时间槽生成、冲突检测、推荐算法）
- ✅ 值对象（BookingStatus、Duration、TechnicianSkill）

**适配**：
- ❌ 去掉聚合根基类（改用CRMEB的Model）
- ❌ 去掉Repository接口（改用CRMEB的Dao）
- ❌ 去掉领域事件（改用Spring Event或MQ）

### 策略2：分层映射

```
DDD预约域                    CRMEB项目
├── Booking聚合根      →    crmeb-common/model/booking/Booking.java
├── TimeSlot实体       →    crmeb-common/model/booking/TimeSlot.java
├── BookingStatus值对象 →    crmeb-common/vo/BookingStatusVo.java
├── BookingRepository  →    crmeb-service/dao/BookingDao.java
├── BookingAppService  →    crmeb-service/service/BookingService.java
└── BookingController  →    crmeb-front/controller/BookingController.java
```

### 第三步：创建Service层（1小时）

**文件**：`crmeb-service/service/impl/BookingServiceImpl.java`

```java
@Service
public class BookingServiceImpl implements BookingService {
    
    @Autowired
    private BookingDao bookingDao;
    
    @Autowired
    private TechnicianScheduleService scheduleService;
    
    @Override
    @Transactional(rollbackFor = Exception.class)
    public Booking createBooking(BookingRequest request) {
        // 1. 参数校验
        validateBookingTime(request.getBookingTime());
        
        // 2. 查找可用时间槽
        TimeSlot timeSlot = scheduleService.findAvailableTimeSlot(
            request.getTechnicianId(),
            request.getBookingTime(),
            request.getDurationMinutes()
        );
        
        if (timeSlot == null) {
            throw new CrmebException("该时间段已被预约");
        }
        
        // 3. 创建预约
        Booking booking = new Booking();
        booking.setBookingNo(generateBookingNo());
        booking.setUserId(request.getUserId());
        booking.setStoreId(request.getStoreId());
        booking.setServiceId(request.getServiceId());
        booking.setBookingTime(request.getBookingTime());
        booking.setDurationMinutes(request.getDurationMinutes());
        booking.setStatus(0); // 待确认
        booking.setCreateTime(new Date());
        
        // 4. 占用时间槽
        boolean occupied = scheduleService.occupyTimeSlot(timeSlot.getId(), booking.getId());
        if (!occupied) {
            throw new CrmebException("时间槽占用失败");
        }
        
        // 5. 保存预约
        bookingDao.insert(booking);
        
        return booking;
    }
    
    private void validateBookingTime(Date bookingTime) {
        Date now = new Date();
        if (bookingTime.before(now)) {
            throw new CrmebException("不能预约过去的时间");
        }
        // 至少提前30分钟
        long diff = bookingTime.getTime() - now.getTime();
        if (diff < 30 * 60 * 1000) {
            throw new CrmebException("请至少提前30分钟预约");
        }
    }
}
```

### 第四步：创建Controller层（30分钟）

**文件**：`crmeb-front/controller/BookingController.java`

```java
@RestController
@RequestMapping("/api/front/booking")
@Api(tags = "预约管理")
public class BookingController {
    
    @Autowired
    private BookingService bookingService;
    
    @ApiOperation("查询可用时间槽")
    @GetMapping("/time-slots/available")
    public CommonResult<List<TimeSlotResponse>> getAvailableTimeSlots(
            @RequestParam Integer technicianId,
            @RequestParam @DateTimeFormat(pattern = "yyyy-MM-dd") Date date,
            @RequestParam Integer durationMinutes) {
        
        List<TimeSlot> timeSlots = bookingService.getAvailableTimeSlots(
            technicianId, date, durationMinutes
        );
        
        return CommonResult.success(timeSlots);
    }
    
    @ApiOperation("创建预约")
    @PostMapping("/create")
    public CommonResult<Booking> createBooking(@RequestBody @Validated BookingRequest request) {
        Booking booking = bookingService.createBooking(request);
        return CommonResult.success(booking);
    }
    
    @ApiOperation("取消预约")
    @PostMapping("/cancel/{bookingNo}")
    public CommonResult<String> cancelBooking(
            @PathVariable String bookingNo,
            @RequestParam String reason) {
        
        bookingService.cancelBooking(bookingNo, reason);
        return CommonResult.success("取消成功");
    }
}
```

```sql
-- 预约表
CREATE TABLE `eb_booking` (
  `id` int(11) NOT NULL AUTO_INCREMENT COMMENT '预约ID',
  `booking_no` varchar(32) NOT NULL COMMENT '预约编号',
  `user_id` int(11) NOT NULL COMMENT '用户ID',
  `store_id` int(11) NOT NULL COMMENT '门店ID',
  `service_id` int(11) NOT NULL COMMENT '服务ID',
  `service_name` varchar(100) DEFAULT NULL COMMENT '服务名称',
  `technician_id` int(11) DEFAULT NULL COMMENT '技师ID',
  `technician_name` varchar(50) DEFAULT NULL COMMENT '技师姓名',
  `technician_skill_level` tinyint(1) DEFAULT NULL COMMENT '技师技能等级',
  `booking_time` datetime NOT NULL COMMENT '预约时间',
  `duration_minutes` int(11) NOT NULL COMMENT '服务时长（分钟）',
  `time_slot_id` int(11) DEFAULT NULL COMMENT '时间槽ID',
  `status` tinyint(1) NOT NULL DEFAULT '0' COMMENT '状态 0-待确认 1-已确认 2-服务中 3-已完成 4-已取消',
  `original_price` decimal(10,2) DEFAULT NULL COMMENT '原价',
  `pay_amount` decimal(10,2) DEFAULT NULL COMMENT '实付金额',
  `order_id` int(11) DEFAULT NULL COMMENT '关联订单ID',
  `remark` varchar(255) DEFAULT NULL COMMENT '备注',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `update_time` datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_booking_no` (`booking_no`),
  KEY `idx_user_id` (`user_id`),
  KEY `idx_technician_time` (`technician_id`, `booking_time`),
  KEY `idx_status` (`status`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='预约表';

-- 时间槽表
CREATE TABLE `eb_time_slot` (
  `id` int(11) NOT NULL AUTO_INCREMENT COMMENT '时间槽ID',
  `technician_id` int(11) NOT NULL COMMENT '技师ID',
  `start_time` datetime NOT NULL COMMENT '开始时间',
  `end_time` datetime NOT NULL COMMENT '结束时间',
  `duration_minutes` int(11) NOT NULL COMMENT '时长（分钟）',
  `is_available` tinyint(1) DEFAULT '1' COMMENT '是否可用',
  `booking_id` int(11) DEFAULT NULL COMMENT '占用的预约ID',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  PRIMARY KEY (`id`),
  KEY `idx_technician_time` (`technician_id`, `start_time`, `end_time`),
  KEY `idx_available` (`is_available`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='时间槽表';
```
