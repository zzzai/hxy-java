### 3. 会员价自动应用

**核心逻辑**:

```php
protected function applyMemberDiscount(float $currentPrice, array $context, array &$priceTrace): array
{
    // 1. 获取用户会员等级
    $memberLevel = $user['level'] ?? 0;
    
    // 2. 获取会员价
    $memberPrice = $storeSku['member_price'] ?? 0;
    
    // 3. 判断是否与闲时价叠加
    $isOverlay = $strategy['is_time_member_overlay'] ?? 0;
    
    if ($isOverlay == 1 && $priceTrace['time_discount'] > 0) {
        // 叠加模式：折扣率相乘
        $memberDiscountRate = bcdiv($memberPrice, $basePrice, 4);
        $finalPrice = bcmul($currentPrice, $memberDiscountRate, 2);
    } else {
        // 非叠加模式：直接使用会员价
        $finalPrice = $memberPrice;
    }
    
    return ['price' => $finalPrice, 'discount' => $discount, ...];
}
```

**支持的叠加策略**:

| 策略 | 配置 | 计算方式 | 示例 |
|------|------|----------|------|
| 不叠加 | `is_time_member_overlay=0` | 后面的价格覆盖前面的 | 闲时88元 → 会员98元 = 98元 |
| 叠加 | `is_time_member_overlay=1` | 折扣率相乘 | 闲时88元 → 会员折扣 = 67.38元 |

## 💡 使用示例

### 场景1: 用户下单前计算价格

```php
use app\services\pricing\PriceCalculatorService;

// 在订单确认页调用
public function orderConfirm()
{
    $priceCalculator = app()->make(PriceCalculatorService::class);
    
    $result = $priceCalculator->calculatePrice(
        $storeId = $this->request->param('store_id'),
        $skuId = $this->request->param('sku_id'),
        $uid = $this->request->uid(),
        $couponInfo = $this->request->param('coupon_info', []),
        $bookingTime = $this->request->param('booking_time', '')
    );
    
    return app('json')->success([
        'original_price' => $result['base_price'],
        'final_price' => $result['final_price'],
        'total_discount' => bcsub($result['base_price'], $result['final_price'], 2),
        'discount_detail' => [
            'time_discount' => $result['time_discount'],
            'member_discount' => $result['member_discount'],
            'coupon_discount' => $result['coupon_discount']
        ],
        'steps' => $result['steps']
    ]);
}
```
