## 🎯 系统概述

### 设计理念

借鉴瑞幸/星巴克的会员体系，核心逻辑：

> **"门槛低、反馈快、复购强"** + **"成长感"** + **"价格歧视"**

### 三大核心策略

| 策略 | 说明 | 借鉴对象 |
|------|------|----------|
| **成长体系** | 1元=1成长值，自动升级 | 星巴克星星进度条 |
| **双价格展示** | 会员价 + 划线原价 | 瑞幸价格锚点 |
| **次卡锁定** | 提前锁定现金流 | 瑞幸咖啡钱包 |

### 会员等级体系（3级）

| 等级 | 名称 | 升级条件 | 折扣 | 权益 |
|------|------|----------|------|------|
| **V0** | 荷芽 | 注册即得 | 无折扣 | 首单半价 |
| **V1** | 荷叶 | 累计消费≥500元 | 9折 | 全场9折+生日礼 |
| **V2** | 荷花 | 累计消费≥2000元 | 8.5折 | 全场8.5折+优先预约+专属客服 |

## ⭐ 核心特性

### 1. 自动升级机制

**触发时机**: 订单核销完成时

**核心逻辑**:
```php
// 订单完成后自动增加成长值并升级
$userLevelService->addGrowthValueAndUpgrade($uid, $orderId, $payPrice);

// 返回结果
[
    'upgraded' => true,
    'old_level' => 0,
    'new_level' => 1,
    'old_growth_value' => 450.00,
    'new_growth_value' => 550.00,
    'growth_added' => 100.00
]
```

**特点**:
- ✅ 成长值基于实付金额（1元=1成长值）
- ✅ 自动升级，无需用户手动操作
- ✅ 只升不降（即使退款也不降级）
- ✅ 自动发送升级通知

---

### 2. 价格计算流水线（集成会员折扣）

**流水线**: `base -> time -> member -> coupon`

**示例**:
```
原价128元 → 闲时88元 → 会员9折79.2元 → 优惠券-10元 = 最终69.2元
```

**核心代码**:
```php
$priceCalculator = app()->make(PriceCalculatorService::class);

$result = $priceCalculator->calculatePrice(
    $storeId,
    $skuId,
    $uid,
    $couponInfo,
    $bookingTime
);

// 返回完整的价格追踪
[
    'base_price' => 128.00,
    'time_discount' => 40.00,
    'member_discount' => 8.80,
    'coupon_discount' => 10.00,
    'final_price' => 69.20,
    'steps' => [...]
]
```

## 🗄️ 数据库设计

### 新增表（6张）

#### 1. `eb_user_level` - 会员等级配置表

```sql
CREATE TABLE `eb_user_level` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `level` int(11) DEFAULT 0 COMMENT '等级编号',
  `name` varchar(32) DEFAULT '' COMMENT '等级名称',
  `growth_required` decimal(10,2) DEFAULT 0.00 COMMENT '升级所需成长值',
  `discount` decimal(4,2) DEFAULT 1.00 COMMENT '会员折扣率',
  `benefits` text COMMENT '会员权益（JSON）',
  PRIMARY KEY (`id`),
  UNIQUE KEY `idx_level` (`level`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='会员等级配置表';
```

#### 2. `eb_newbie_offer` - 新人引流品配置表

```sql
CREATE TABLE `eb_newbie_offer` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `service_id` int(11) DEFAULT 0 COMMENT '关联服务ID',
  `original_price` decimal(10,2) DEFAULT 0.00 COMMENT '原价',
  `newbie_price` decimal(10,2) DEFAULT 0.00 COMMENT '新人价',
  `daily_limit` int(11) DEFAULT 0 COMMENT '每日限量',
  `user_limit` int(11) DEFAULT 1 COMMENT '每人限购次数',
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='新人引流品配置表';
```

#### 3. `eb_gift_card` - 次卡/礼品卡表

```sql
CREATE TABLE `eb_gift_card` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `card_no` varchar(32) DEFAULT '' COMMENT '卡号',
  `card_type` tinyint(1) DEFAULT 1 COMMENT '卡类型: 1=次卡 2=储值卡',
  `value` decimal(10,2) DEFAULT 0.00 COMMENT '面值',
  `balance` decimal(10,2) DEFAULT 0.00 COMMENT '余额',
  `expire_time` int(11) DEFAULT 0 COMMENT '过期时间',
  PRIMARY KEY (`id`),
  UNIQUE KEY `idx_card_no` (`card_no`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='次卡/礼品卡表';
```

### 第三步：集成到业务代码（30分钟）

#### 1. 订单完成时增加成长值

```php
// 在订单核销完成的回调中
use app\services\user\UserLevelService;

public function orderComplete($orderId)
{
    $order = Order::find($orderId);
    
    $userLevelService = app()->make(UserLevelService::class);
    
    // 增加成长值并自动升级
    $result = $userLevelService->addGrowthValueAndUpgrade(
        $order['uid'],
        $orderId,
        $order['pay_price']
    );
    
    if ($result['upgraded']) {
        // 发送升级通知
        $this->sendUpgradeNotice($order['uid'], $result);
    }
}
```

#### 2. 下单前计算价格

```php
// 在订单确认页
use app\services\pricing\PriceCalculatorService;

public function orderConfirm()
{
    $priceCalculator = app()->make(PriceCalculatorService::class);
    
    $priceResult = $priceCalculator->calculatePrice(
        $this->request->param('store_id'),
        $this->request->param('sku_id'),
        $this->request->uid(),
        $this->request->param('coupon_info', []),
        $this->request->param('booking_time', '')
    );
    
    return app('json')->success($priceResult);
}
```

#### 3. 展示新人引流品

```php
// 在首页或服务列表页
use app\services\activity\NewbieOfferService;

public function getNewbieOffers()
{
    $newbieOfferService = app()->make(NewbieOfferService::class);
```
