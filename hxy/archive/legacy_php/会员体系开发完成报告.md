## 🎯 核心功能详解

### 1. 会员等级体系（3级）

**设计理念**: 借鉴瑞幸/星巴克，极简3级体系

| 等级 | 名称 | 升级条件 | 折扣 | 权益 |
|------|------|----------|------|------|
| **V0 荷芽** | 新客 | 注册即得 | 无折扣 | 首单半价 |
| **V1 荷叶** | 熟客 | 累计消费≥500元 | 9折 | 全场9折+生日礼 |
| **V2 荷花** | 铁粉 | 累计消费≥2000元 | 8.5折 | 全场8.5折+优先预约+专属客服 |

**核心特点**:
- ✅ 成长值基于实付金额（1元=1成长值）
- ✅ 自动升级，无需用户手动操作
- ✅ 只升不降（即使退款也不降级）
- ✅ 自动发送升级通知

**核心代码**:
```php

    'upgraded' => true,
```

### 2. 价格计算流水线（集成会员折扣）

**流水线**: `base -> time -> member -> coupon`

**示例计算**:
```
原价128元 
→ 闲时优惠 -40元 = 88元
→ V1会员9折 -7.92元 = 80.08元
→ 优惠券 -10元 = 70.08元
```

**核心代码**:
```php
protected function applyMemberDiscount(float $currentPrice, array $context, array &$priceTrace): array
{
    $user = $context['user'];
    $memberLevel = $user['level'] ?? 0;

    // 获取等级配置
    $levelConfig = UserLevelService::LEVEL_CONFIG[$memberLevel] ?? null;

    if (!$levelConfig || $levelConfig['discount'] >= 1.0) {
        return ['price' => $currentPrice, 'discount' => 0, 'description' => '非会员用户'];
    }

    // 应用会员折扣
    $finalPrice = bcmul($currentPrice, $levelConfig['discount'], 2);
    $discount = bcsub($currentPrice, $finalPrice, 2);

    return [
        'price' => $finalPrice,
        'discount' => $discount,
        'description' => "{$levelConfig['name']}优惠（{$discountPercent}折）: -¥{$discount}"
    ];
}
```
