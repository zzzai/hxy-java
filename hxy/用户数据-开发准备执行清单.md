## 4. 联调优先级（按顺序）

1. 前台授权链路：`grant -> list -> withdraw`
2. 前台删除链路：`request -> list -> cancel`
3. 管理工单链路：`create -> approve/reject -> close`
4. 标签上线拦截规则链路：`label-policy/list -> update-status`

## 5. 验收门槛

1. 无授权写入健康数据应被拦截。
2. L3/L4访问必须走工单审批。
3. 删除工单应进入7天冷静期。
4. 高争议标签默认不自动决策。
