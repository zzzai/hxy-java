# 补全模块5：会员储值卡跨店使用

**优先级**：P2（建议）  
**影响范围**：会员体系、跨店结算、用户体验  
**开发工作量**：1.5天

---

## 一、问题描述

原方案未提及**储值卡跨店使用和结算**，导致：

### 业务痛点

1. **储值卡使用范围不清**
   - 用户在A店充值1000元，能在B店消费吗？
   - 如果能用，收入怎么结算？

2. **门店利益冲突**
   - A店花成本做充值活动，用户去B店消费
   - A店承担成本，B店获得收益，不公平

3. **财务结算困难**
   - 跨店消费后，如何结算？
   - 实时结算还是月结？

4. **充值优惠问题**
   - A店充1000送200，用户去B店消费
   - B店是否认可这200元赠送？

---

## 二、解决方案

### 方案核心：储值卡类型 + 跨店结算 + 实时清算 + 赠送金分离

## 三、储值卡类型设计

### 3.1 储值卡分类

| 卡类型 | 使用范围 | 充值方 | 结算方式 | 示例 |
|--------|----------|---------|----------|------|
| **单店卡** | 仅限充值门店 | 门店 | 无需结算 | A店充值，仅A店用 |
| **全国通用卡** | 所有门店 | 总部 | 总部承担 | 总部发卡，全国通用 |
| **跨店通用卡** | 所有门店 | 门店 | 实时结算 | A店充值，B店消费，实时结算 |

---

### 3.2 储值金类型

| 金额类型 | 说明 | 跨店使用 | 结算规则 |
|---------|------|----------|----------|
| **本金** | 用户实际充值金额 | 可跨店 | 按实际消费结算 |
| **赠送金** | 充值活动赠送金额 | 可跨店 | 充值门店承担 |
| **总部赠送** | 总部营销赠送 | 可跨店 | 总部承担 |

```php
// app/services/balance/BalanceCardService.php

public function recharge($userId, $storeId, $amount, $giftAmount = 0)
{
    // 1. 创建或更新储值卡
    $card = Db::name('user_balance_card')
        ->where('user_id', $userId)
        ->where('issue_store_id', $storeId)
        ->where('card_type', 3)  // 跨店通用卡
        ->find();
    
    if (!$card) {
        // 创建新卡
        $cardId = Db::name('user_balance_card')->insertGetId([
            'user_id' => $userId,
            'card_type' => 3,
            'issue_store_id' => $storeId,
            'total_amount' => $amount + $giftAmount,
            'principal_amount' => $amount,
            'gift_amount' => $giftAmount,
            'status' => 1,
            'create_time' => time()
        ]);
    } else {
        // 更新余额
        Db::name('user_balance_card')
            ->where('id', $card['id'])
            ->update([
                'total_amount' => Db::raw('total_amount + ' . ($amount + $giftAmount)),
                'principal_amount' => Db::raw('principal_amount + ' . $amount),
                'gift_amount' => Db::raw('gift_amount + ' . $giftAmount)
            ]);
        $cardId = $card['id'];
    }
    
    // 2. 记录流水
    Db::name('balance_card_flow')->insert([
        'card_id' => $cardId,
        'user_id' => $userId,
        'change_type' => 1,  // 充值
        'change_amount' => $amount + $giftAmount,
        'principal_change' => $amount,
        'gift_change' => $giftAmount,
        'before_amount' => $card['total_amount'] ?? 0,
        'after_amount' => ($card['total_amount'] ?? 0) + $amount + $giftAmount,
        'store_id' => $storeId,
        'remark' => '充值' . $amount . '元' . ($giftAmount > 0 ? '，赠送' . $giftAmount . '元' : ''),
        'create_time' => time()
    ]);
    
    return $cardId;
}
```

### 5.2 跨店消费

```php
public function consume($userId, $storeId, $orderId, $amount)
{
    // 1. 获取用户储值卡（优先使用跨店通用卡）
    $card = Db::name('user_balance_card')
        ->where('user_id', $userId)
        ->where('status', 1)
        ->where(function($query) use ($storeId) {
            $query->where('card_type', 'in', [2, 3])  // 全国通用或跨店通用
                  ->whereOr(function($q) use ($storeId) {
                      $q->where('card_type', 1)
                        ->where('issue_store_id', $storeId);  // 或本店单店卡
                  });
        })
        ->order('card_type', 'desc')  // 优先使用跨店通用卡
        ->find();
    
    if (!$card) {
        throw new Exception('没有可用的储值卡');
    }
    
    if ($card['total_amount'] < $amount) {
        throw new Exception('储值卡余额不足');
    }
    
    Db::startTrans();
    try {
        // 2. 扣减余额（优先扣赠送金）
        $giftDeduct = min($card['gift_amount'], $amount);
        $principalDeduct = $amount - $giftDeduct;
        
        Db::name('user_balance_card')
            ->where('id', $card['id'])
            ->update([
                'total_amount' => Db::raw('total_amount - ' . $amount),
                'principal_amount' => Db::raw('principal_amount - ' . $principalDeduct),
                'gift_amount' => Db::raw('gift_amount - ' . $giftDeduct),
                'used_amount' => Db::raw('used_amount + ' . $amount)
            ]);
        
        // 3. 记录流水
        Db::name('balance_card_flow')->insert([
            'card_id' => $card['id'],
            'user_id' => $userId,
            'change_type' => 2,  // 消费
            'change_amount' => -$amount,
            'principal_change' => -$principalDeduct,
            'gift_change' => -$giftDeduct,
            'before_amount' => $card['total_amount'],
            'after_amount' => $card['total_amount'] - $amount,
            'store_id' => $storeId,
            'order_id' => $orderId,
            'remark' => '订单消费',
            'create_time' => time()
        ]);
        
        // 4. 如果是跨店消费，创建结算记录
        if ($card['issue_store_id'] != $storeId && $card['card_type'] == 3) {
            $this->createSettlement(
                $orderId,
                $card['id'],
                $userId,
                $card['issue_store_id'],
                $storeId,
                $amount,
                $principalDeduct,
                $giftDeduct
            );
        }
        
        Db::commit();
        
        return [
            'success' => true,
            'remaining' => $card['total_amount'] - $amount
        ];
        
    } catch (Exception $e) {
        Db::rollback();
        throw $e;
    }
}
```

### 5.3 跨店结算

```php
private function createSettlement($orderId, $cardId, $userId, $issueStoreId, 
    $useStoreId, $consumeAmount, $principalAmount, $giftAmount)
{
    // 1. 计算结算金额
    // 本金：消费门店获得100%
    // 赠送金：发卡门店承担100%
    
    $issueStoreCost = $giftAmount;  // 发卡门店承担赠送金
    $useStoreIncome = $consumeAmount;  // 消费门店获得全部消费金额
    
    // 2. 创建结算记录
    Db::name('balance_settlement')->insert([
        'order_id' => $orderId,
        'card_id' => $cardId,
        'user_id' => $userId,
        'issue_store_id' => $issueStoreId,
        'use_store_id' => $useStoreId,
        'consume_amount' => $consumeAmount,
        'principal_amount' => $principalAmount,
        'gift_amount' => $giftAmount,
        'issue_store_cost' => $issueStoreCost,
        'use_store_income' => $useStoreIncome,
        'settlement_status' => 0,  // 待结算
        'create_time' => time()
    ]);
}

// 月度结算
public function monthlySettlement($month)
{
    // 1. 查询待结算记录
    $settlements = Db::name('balance_settlement')
        ->where('settlement_status', 0)
        ->where('create_time', '>=', strtotime($month . '-01'))
        ->where('create_time', '<', strtotime($month . '-01 +1 month'))
        ->select();
    
    // 2. 按门店汇总
    $storeStats = [];
    
    foreach ($settlements as $settlement) {
        // 发卡门店统计
        if (!isset($storeStats[$settlement['issue_store_id']])) {
            $storeStats[$settlement['issue_store_id']] = [
                'issue_cost' => 0,
                'use_income' => 0
            ];
        }
        $storeStats[$settlement['issue_store_id']]['issue_cost'] += $settlement['issue_store_cost'];
        
        // 消费门店统计
        if (!isset($storeStats[$settlement['use_store_id']])) {
            $storeStats[$settlement['use_store_id']] = [
                'issue_cost' => 0,
                'use_income' => 0
            ];
        }
        $storeStats[$settlement['use_store_id']]['use_income'] += $settlement['use_store_income'];
        
        // 更新结算状态
        Db::name('balance_settlement')
            ->where('id', $settlement['id'])
            ->update([
                'settlement_status' => 1,
                'settlement_time' => time()
            ]);
    }
    
    return $storeStats;
}
```

## 九、总结

### 核心价值
1. **跨店通用**：用户可在任意门店消费
2. **结算清晰**：本金和赠送金分离计算
3. **利益平衡**：发卡门店承担赠送金成本
4. **财务透明**：月度自动结算

### 关键规则
- ✅ 本金可跨店使用，消费门店获得收入
- ✅ 赠送金可跨店使用，发卡门店承担成本
- ✅ 优先扣除赠送金
- ✅ 月度自动结算
