# 补全模块13：缓存策略分层设计

**优先级**：P2（建议）  
**影响范围**：系统性能、响应速度、数据库压力  
**开发工作量**：1天

---

## 一、问题描述

原方案提到"避免N+1查询"，但**未说明缓存方案**，导致：

### 业务痛点

1. **数据库压力大**
   - 每次查询都直接访问数据库
   - 高并发时数据库成为瓶颈

2. **响应速度慢**
   - 总部主数据查询频繁
   - 门店SKU价格查询慢

3. **重复查询多**
   - 同一个SKU被多次查询
   - 技师排班数据重复加载

4. **缓存策略不清**
   - 不知道哪些数据该缓存
   - 不知道缓存多久
   - 不知道何时更新缓存

---

## 二、解决方案

### 方案核心：分层缓存 + 主动更新 + 过期策略 + 缓存预热

---

## 三、缓存分层设计

### 3.1 缓存层级

| 层级 | 缓存位置 | 适用数据 | 过期时间 | 更新方式 |
|------|----------|----------|----------|----------|
| **L1：本地缓存** | PHP进程内存 | 请求内复用数据 | 请求结束 | 自动清除 |
| **L2：Redis缓存** | Redis | 热点数据 | 5分钟-1小时 | 主动更新 |
| **L3：数据库** | MySQL | 持久化数据 | 永久 | 实时更新 |

---

### 3.2 缓存数据分类

| 数据类型 | 缓存策略 | 过期时间 | 更新时机 | 示例 |
|---------|---------|---------|---------|------|
| **总部主数据** | Redis缓存 | 1小时 | 主动更新 | SPU/SKU基础信息 |
| **门店运营数据** | Redis缓存 | 15分钟 | 写入时更新 | 门店SKU价格、库存 |
| **价格冗余** | 订单表冗余 | 永久 | 下单时写入 | 订单价格快照 |
| **技师排班** | Redis缓存 | 当天23:59 | 生成时写入 | 当日排班数据 |
| **用户会员信息** | Redis缓存 | 30分钟 | 变更时更新 | 用户等级、折扣率 |
| **优惠券** | Redis缓存 | 10分钟 | 使用时更新 | 用户可用券列表 |

## 四、缓存实现

### 4.1 Redis缓存封装

```php
// app/services/cache/CacheService.php

namespace app\services\cache;

use think\facade\Cache;

class CacheService
{
    /**
     * 缓存key前缀
     */
    const PREFIX = 'hxy:';
    
    /**
     * 缓存过期时间（秒）
     */
    const TTL_SHORT = 300;      // 5分钟
    const TTL_MEDIUM = 900;     // 15分钟
    const TTL_LONG = 3600;      // 1小时
    const TTL_DAY = 86400;      // 1天
    
    /**
     * 获取缓存
     */
    public static function get($key, $default = null)
    {
        $fullKey = self::PREFIX . $key;
        $value = Cache::get($fullKey);
        
        return $value !== false ? $value : $default;
    }
    
    /**
     * 设置缓存
     */
    public static function set($key, $value, $ttl = self::TTL_MEDIUM)
    {
        $fullKey = self::PREFIX . $key;
        return Cache::set($fullKey, $value, $ttl);
    }
    
    /**
     * 删除缓存
     */
    public static function delete($key)
    {
        $fullKey = self::PREFIX . $key;
        return Cache::delete($fullKey);
    }
    
    /**
     * 批量删除（按前缀）
     */
    public static function deleteByPrefix($prefix)
    {
        $fullPrefix = self::PREFIX . $prefix;
        $keys = Cache::keys($fullPrefix . '*');
        
        foreach ($keys as $key) {
            Cache::delete($key);
        }
    }
    
    /**
     * 记住缓存（如果不存在则执行回调并缓存）
     */
    public static function remember($key, $callback, $ttl = self::TTL_MEDIUM)
    {
        $value = self::get($key);
        
        if ($value !== null) {
            return $value;
        }
        
        $value = $callback();
        self::set($key, $value, $ttl);
        
        return $value;
    }
}
```

### 4.2 总部主数据缓存

```php
// app/services/cache/SkuCacheService.php

class SkuCacheService
{
    /**
     * 获取SKU信息（带缓存）
     */
    public static function getSku($skuId)
    {
        $cacheKey = "sku:info:{$skuId}";
        
        return CacheService::remember($cacheKey, function() use ($skuId) {
            return Db::name('service_sku')->where('id', $skuId)->find();
        }, CacheService::TTL_LONG);
    }
    
    /**
     * 批量获取SKU（带缓存）
     */
    public static function getSkuList($skuIds)
    {
        $result = [];
        $missIds = [];
        
        // 1. 先从缓存获取
        foreach ($skuIds as $skuId) {
            $cacheKey = "sku:info:{$skuId}";
            $sku = CacheService::get($cacheKey);
            
            if ($sku) {
                $result[$skuId] = $sku;
            } else {
                $missIds[] = $skuId;
            }
        }
        
        // 2. 缓存未命中的从数据库查询
        if (!empty($missIds)) {
            $skus = Db::name('service_sku')
                ->where('id', 'in', $missIds)
                ->select();
            
            foreach ($skus as $sku) {
                $cacheKey = "sku:info:{$sku['id']}";
                CacheService::set($cacheKey, $sku, CacheService::TTL_LONG);
                $result[$sku['id']] = $sku;
            }
        }
        
        return $result;
    }
    
    /**
     * 更新SKU时清除缓存
     */
    public static function clearSkuCache($skuId)
    {
        $cacheKey = "sku:info:{$skuId}";
        CacheService::delete($cacheKey);
        
        // 同时清除门店SKU缓存
        CacheService::deleteByPrefix("store_sku:sku:{$skuId}");
    }
}
```

// app/services/cache/StoreSkuCacheService.php

class StoreSkuCacheService
{
    /**
     * 获取门店SKU（带缓存）
     */
    public static function getStoreSku($storeId, $skuId)
    {
        $cacheKey = "store_sku:{$storeId}:{$skuId}";
        
        return CacheService::remember($cacheKey, function() use ($storeId, $skuId) {
            return Db::name('store_service_sku')
                ->where('store_id', $storeId)
                ->where('sku_id', $skuId)
                ->find();
        }, CacheService::TTL_MEDIUM);
    }
    
    /**
     * 获取门店所有SKU（带缓存）
     */
    public static function getStoreSkuList($storeId)
    {
        $cacheKey = "store_sku:list:{$storeId}";
        
        return CacheService::remember($cacheKey, function() use ($storeId) {
            return Db::name('store_service_sku')
                ->where('store_id', $storeId)
                ->where('status', 1)
                ->select();
        }, CacheService::TTL_MEDIUM);
    }
    
    /**
     * 更新门店SKU时清除缓存
     */
    public static function clearStoreSkuCache($storeId, $skuId = null)
    {
        if ($skuId) {
            // 清除单个SKU缓存
            $cacheKey = "store_sku:{$storeId}:{$skuId}";
            CacheService::delete($cacheKey);
        }
        
        // 清除门店SKU列表缓存
        $listKey = "store_sku:list:{$storeId}";
        CacheService::delete($listKey);
    }
}
```

// app/services/cache/ScheduleCacheService.php

class ScheduleCacheService
{
    /**
     * 获取技师排班（带缓存）
     */
    public static function getSchedule($storeId, $skuId, $date)
    {
        $cacheKey = "schedule:{$storeId}:{$skuId}:{$date}";
        
        // 当天排班缓存到23:59
        $expireTime = strtotime($date . ' 23:59:59') - time();
        $ttl = max($expireTime, 60);  // 最少缓存1分钟
        
        return CacheService::remember($cacheKey, function() use ($storeId, $skuId, $date) {
            return Db::name('technician_schedule')
                ->where('store_id', $storeId)
                ->where('service_sku_id', $skuId)
                ->where('work_date', $date)
                ->where('status', 1)  // 可预约
                ->select();
        }, $ttl);
    }
    
    /**
     * 预约时清除缓存
     */
    public static function clearScheduleCache($storeId, $skuId, $date)
    {
        $cacheKey = "schedule:{$storeId}:{$skuId}:{$date}";
        CacheService::delete($cacheKey);
    }
}
```

## 五、缓存更新策略

### 5.1 主动更新（推荐）

```php
// 数据变更时主动更新缓存

public function updateStoreSku($id, $data)
{
    // 1. 更新数据库
    Db::name('store_service_sku')->where('id', $id)->update($data);
    
    // 2. 清除缓存
    $sku = Db::name('store_service_sku')->where('id', $id)->find();
    StoreSkuCacheService::clearStoreSkuCache($sku['store_id'], $sku['sku_id']);
    
    // 3. 可选：立即写入新缓存
    StoreSkuCacheService::getStoreSku($sku['store_id'], $sku['sku_id']);
}
```

### 5.2 延迟双删（防止缓存不一致）

```php
public function updateStoreSkuWithDelayDelete($id, $data)
{
    $sku = Db::name('store_service_sku')->where('id', $id)->find();
    
    // 1. 第一次删除缓存
    StoreSkuCacheService::clearStoreSkuCache($sku['store_id'], $sku['sku_id']);
    
    // 2. 更新数据库
    Db::name('store_service_sku')->where('id', $id)->update($data);
    
    // 3. 延迟500ms后再次删除缓存（防止并发读取到旧数据）
    usleep(500000);
    StoreSkuCacheService::clearStoreSkuCache($sku['store_id'], $sku['sku_id']);
}
```

// app/command/CacheWarmup.php

protected function execute(Input $input, Output $output)
{
    $output->writeln('开始缓存预热...');
    
    // 1. 预热总部SKU数据

    // 2. 预热门店SKU数据（热门门店）

    foreach ($hotStores as $storeId) {

            ScheduleCacheService::getSchedule($storeId, $skuId, $today);
        }
    }
    $output->writeln('排班数据预热完成');
    
    $output->writeln('缓存预热完成！');

## 七、缓存监控

### 7.1 缓存命中率统计

```php
// app/services/cache/CacheMonitor.php

class CacheMonitor
{
    /**
     * 记录缓存命中
     */
    public static function hit($key)
    {
        $statsKey = 'cache:stats:hit:' . date('Ymd');
        Cache::inc($statsKey);
        Cache::expire($statsKey, 86400 * 7);  // 保留7天
    }
    
    /**
     * 记录缓存未命中
     */
    public static function miss($key)
    {
        $statsKey = 'cache:stats:miss:' . date('Ymd');
        Cache::inc($statsKey);
        Cache::expire($statsKey, 86400 * 7);
    }
    
    /**
     * 获取缓存命中率
     */
    public static function getHitRate($date = null)
    {
        $date = $date ?: date('Ymd');
        
        $hit = Cache::get('cache:stats:hit:' . $date) ?: 0;
        $miss = Cache::get('cache:stats:miss:' . $date) ?: 0;
        $total = $hit + $miss;
        
        if ($total == 0) {
            return 0;
        }
        
        return round($hit / $total * 100, 2);
    }
}
```

## 十、注意事项

### 10.1 缓存穿透

**问题**：查询不存在的数据，缓存和数据库都没有

**解决**：缓存空值

```php
public static function getSku($skuId)
{
    $cacheKey = "sku:info:{$skuId}";
    
    return CacheService::remember($cacheKey, function() use ($skuId) {
        $sku = Db::name('service_sku')->where('id', $skuId)->find();
        
        // 如果不存在，缓存空值（过期时间短）
        return $sku ?: ['_empty' => true];
    }, CacheService::TTL_SHORT);
}
```

### 10.2 缓存雪崩

**问题**：大量缓存同时过期，数据库压力骤增

**解决**：过期时间加随机值

```php
$ttl = CacheService::TTL_MEDIUM + rand(0, 300);  // 15分钟 + 0-5分钟随机
CacheService::set($key, $value, $ttl);
```

### 10.3 缓存击穿

**问题**：热点数据过期瞬间，大量请求打到数据库

**解决**：互斥锁

```php
public static function getSku($skuId)
{
    $cacheKey = "sku:info:{$skuId}";
    $value = CacheService::get($cacheKey);
    
    if ($value !== null) {
        return $value;
    }
    
    // 获取锁
    $lockKey = "lock:sku:{$skuId}";
    $lock = Cache::set($lockKey, 1, ['NX', 'EX' => 10]);
    
    if ($lock) {
        // 获取锁成功，查询数据库
        $value = Db::name('service_sku')->where('id', $skuId)->find();
        CacheService::set($cacheKey, $value, CacheService::TTL_LONG);
        Cache::delete($lockKey);
        return $value;
    } else {
        // 获取锁失败，等待100ms后重试
        usleep(100000);
        return self::getSku($skuId);
    }
}
```
