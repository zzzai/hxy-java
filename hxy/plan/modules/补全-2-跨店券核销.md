# 补全模块2：跨门店通用券核销

**优先级**：P1（重要）  
**影响范围**：营销系统、门店结算、用户体验  
**开发工作量**：1.5天

---

## 一、问题描述

原方案提到"总部发券"，但**未说明跨店核销逻辑**，导致：

### 业务痛点

1. **券的使用范围不清**
   - 用户在A店领券，能在B店用吗？
   - 总部发的全国通用券，所有门店都能用吗？

2. **收入分配不明确**
   - 用户在A店领券，在B店使用
   - 折扣损失由谁承担？A店？B店？总部？

3. **门店利益冲突**
   - A店花钱做活动发券，用户去B店消费
   - A店承担成本，B店获得收益，不公平

4. **财务结算困难**
   - 跨店券使用后，如何结算？
   - 总部补贴如何计算？

---

## 二、解决方案

### 方案核心：券的作用域 + 成本承担方 + 自动结算

## 三、券的分类体系

### 3.1 按作用域分类

| 券类型 | 作用域 | 发放方 | 成本承担方 | 使用场景 |
|--------|--------|--------|------------|----------|
| **单店券** | 仅限发放门店使用 | 门店 | 门店 | 门店自主营销 |
| **门店组券** | 指定门店组可用 | 总部/门店组 | 总部/门店组 | 区域联合营销 |
| **全国通用券** | 所有门店可用 | 总部 | 总部 | 品牌推广活动 |
| **跨店通用券** | 所有门店可用 | 门店A | 总部补贴 | 门店引流，总部兜底 |

### 5.2 成本分摊规则

#### 场景1：单店券（scope_type=1）

```
发放门店：A店
使用门店：A店
成本承担：A店 100%
```

#### 场景2：全国通用券（scope_type=3）

```
发放方：总部
使用门店：任意门店
成本承担：总部 100%
```

#### 场景3：跨店通用券（scope_type=4）

```
发放门店：A店
使用门店：B店
成本承担：总部 100%（鼓励门店发券引流）
```

#### 场景4：门店组券（scope_type=2，按比例分摊）

```
发放方：总部
使用门店：B店
优惠金额：¥50
成本分摊比例：{"issue":0,"use":0.3,"hq":0.7}

计算：
- 发放门店承担：¥50 × 0 = ¥0
- 使用门店承担：¥50 × 0.3 = ¥15
- 总部承担：¥50 × 0.7 = ¥35
```

## 六、代码实现示例

### 6.1 券的使用校验

```php
// app/services/coupon/CouponService.php

public function validateCoupon($couponUserId, $storeId, $orderAmount)
{
    // 1. 获取用户券信息
    $couponUser = Db::name('coupon_user')
        ->alias('cu')
        ->join('coupon c', 'cu.coupon_id = c.id')
        ->where('cu.id', $couponUserId)
        ->where('cu.status', 0)  // 未使用
        ->field('cu.*, c.scope_type, c.scope_store_ids, c.issue_store_id, c.cost_bearer, c.cost_ratio')
        ->find();
    
    if (!$couponUser) {
        throw new Exception('优惠券不存在或已使用');
    }
    
    // 2. 校验作用域
    switch ($couponUser['scope_type']) {
        case 1:  // 单店券
            if ($storeId != $couponUser['issue_store_id']) {
                throw new Exception('该券仅限' . $this->getStoreName($couponUser['issue_store_id']) . '使用');
            }
            break;
            
        case 2:  // 门店组券
            $storeIds = json_decode($couponUser['scope_store_ids'], true);
            if (!in_array($storeId, $storeIds)) {
                throw new Exception('该券不适用于当前门店');
            }
            break;
            
        case 3:  // 全国通用券
        case 4:  // 跨店通用券
            // 所有门店可用
            break;
            
        default:
            throw new Exception('优惠券类型错误');
    }
    
    // 3. 校验订单金额
    if ($orderAmount < $couponUser['use_min_price']) {
        throw new Exception('订单金额不满足使用条件');
    }
    
    // 4. 校验有效期
    if (time() > $couponUser['end_time']) {
        throw new Exception('优惠券已过期');
    }
    
    return true;
}
```

### 6.2 使用券并计算成本分摊

```php
public function useCoupon($couponUserId, $orderId, $storeId, $orderAmount)
{
    // 1. 获取券信息
    $couponUser = Db::name('coupon_user')
        ->alias('cu')
        ->join('coupon c', 'cu.coupon_id = c.id')
        ->where('cu.id', $couponUserId)
        ->field('cu.*, c.scope_type, c.issue_store_id, c.cost_bearer, c.cost_ratio, c.coupon_price')
        ->find();
    
    // 2. 计算优惠金额
    $discountAmount = $couponUser['coupon_price'];
    
    // 3. 判断是否跨店使用
    $isCrossStore = ($couponUser['issue_store_id'] != $storeId) ? 1 : 0;
    
    // 4. 更新用户券状态
    Db::name('coupon_user')->where('id', $couponUserId)->update([
        'status' => 1,  // 已使用
        'use_time' => time(),
        'use_store_id' => $storeId,
        'is_cross_store' => $isCrossStore,
        'discount_amount' => $discountAmount
    ]);
    
    // 5. 如果是跨店使用，计算成本分摊
    if ($isCrossStore) {
        $this->calculateCrossStoreCost(
            $orderId,
            $couponUserId,
            $couponUser['issue_store_id'],
            $storeId,
            $discountAmount,
            $couponUser['cost_bearer'],
            $couponUser['cost_ratio']
        );
    }
    
    return $discountAmount;
}

private function calculateCrossStoreCost($orderId, $couponUserId, $issueStoreId, 
    $useStoreId, $discountAmount, $costBearer, $costRatio)
{
    $issueStoreCost = 0;
    $useStoreCost = 0;
    $hqCost = 0;
    
    switch ($costBearer) {
        case 1:  // 发放门店承担
            $issueStoreCost = $discountAmount;
            break;
            
        case 2:  // 使用门店承担
            $useStoreCost = $discountAmount;
            break;
            
        case 3:  // 总部承担
            $hqCost = $discountAmount;
            break;
            
        case 4:  // 按比例分摊
            $ratio = json_decode($costRatio, true);
            $issueStoreCost = $discountAmount * ($ratio['issue'] ?? 0);
            $useStoreCost = $discountAmount * ($ratio['use'] ?? 0);
            $hqCost = $discountAmount * ($ratio['hq'] ?? 0);
            break;
    }
    
    // 记录结算表
    Db::name('cross_store_settlement')->insert([
        'order_id' => $orderId,
        'coupon_user_id' => $couponUserId,
        'issue_store_id' => $issueStoreId,
        'use_store_id' => $useStoreId,
        'discount_amount' => $discountAmount,
        'issue_store_cost' => $issueStoreCost,
        'use_store_cost' => $useStoreCost,
        'hq_cost' => $hqCost,
        'settlement_status' => 0,
        'create_time' => time()
    ]);
}
```
