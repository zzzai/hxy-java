# 补全模块6：价格体系多层级设计

**优先级**：P1（重要）  
**影响范围**：定价策略、会员体系、营销活动  
**开发工作量**：1天

---

## 一、问题描述

原方案提到"会员价"和"闲时优惠"，但**价格层级不够清晰**，导致：

### 业务痛点

1. **价格层级混乱**
   - 只有"指导价"和"会员价"，层级太少
   - VIP用户、高级会员的差异化定价无法实现

2. **营销活动受限**
   - 无法针对不同用户群体设置不同价格
   - 限时促销、新人专享等场景无法支持

3. **前端展示困难**
   - 不知道该显示哪个价格
   - 划线价、到手价、会员价混乱

4. **价格优先级不清**
   - 用户同时是会员+使用优惠券+闲时优惠
   - 哪个价格优先？如何叠加？

---

## 二、解决方案

### 方案核心：6层价格体系 + 优先级规则 + 前端智能展示

## 三、价格体系设计

### 3.1 价格层级定义

| 层级 | 价格名称 | 适用对象 | 优先级 | 说明 |
|------|----------|----------|--------|------|
| **1** | 原价 | 所有用户 | 最低 | 门店设置的基础价格 |
| **2** | 会员价 | 普通会员 | 低 | 会员专享价格（如9折） |
| **3** | VIP价 | VIP会员 | 中 | VIP专享价格（如8折） |
| **4** | 闲时价 | 所有用户 | 中高 | 非高峰时段优惠价 |
| **5** | 活动价 | 参与活动用户 | 高 | 限时促销、新人专享等 |
| **6** | 优惠券 | 持券用户 | 最高 | 在最终价格基础上再减 |

---

### 3.2 价格计算优先级

```
1. 确定基础价格
   原价 → 会员价 → VIP价 → 闲时价 → 活动价
   （取优先级最高的）

2. 应用优惠券
   基础价格 - 优惠券金额 = 最终价格

3. 前端展示
   划线价：原价
   到手价：最终价格
   优惠标签：显示使用的价格类型
```

## 五、价格计算逻辑

### 5.1 价格计算函数

```php
// app/services/price/PriceService.php

public function calculatePrice($storeSkuId, $userId, $serviceTime = 0)
{
    // 1. 获取SKU价格信息
    $storeSku = Db::name('store_service_sku')
        ->where('id', $storeSkuId)
        ->find();
    
    // 2. 获取用户会员信息
    $userLevel = Db::name('user_level')
        ->where('user_id', $userId)
        ->where('expire_time', '>', time())
        ->find();
    
    // 3. 确定基础价格（按优先级）
    $basePrice = $this->getBasePrice($storeSku, $userLevel, $serviceTime);
    
    // 4. 返回价格信息
    return [
        'original_price' => $storeSku['original_price'],  // 原价
        'base_price' => $basePrice['price'],  // 基础价格
        'price_type' => $basePrice['type'],  // 价格类型
        'price_label' => $basePrice['label'],  // 价格标签
        'discount_amount' => $storeSku['original_price'] - $basePrice['price'],  // 优惠金额
        'can_use_coupon' => true  // 是否可用券
    ];
}

private function getBasePrice($storeSku, $userLevel, $serviceTime)
{
    $now = time();
    
    // 优先级5：活动价（限时）
    if ($storeSku['activity_price'] > 0 
        && $now >= $storeSku['activity_start_time'] 
        && $now <= $storeSku['activity_end_time']) {
        return [
            'price' => $storeSku['activity_price'],
            'type' => 'activity',
            'label' => '限时特惠'
        ];
    }
    
    // 优先级4：闲时价（指定时间段）
    if ($storeSku['offpeak_price'] > 0 && $this->isOffpeakTime($serviceTime)) {
        return [
            'price' => $storeSku['offpeak_price'],
            'type' => 'offpeak',
            'label' => '闲时优惠'
        ];
    }
    
    // 优先级3：VIP价
    if ($userLevel && $userLevel['level_type'] == 2 && $storeSku['vip_price'] > 0) {
        return [
            'price' => $storeSku['vip_price'],
            'type' => 'vip',
            'label' => 'VIP专享'
        ];
    }
    
    // 优先级2：会员价
    if ($userLevel && $userLevel['level_type'] >= 1 && $storeSku['member_price'] > 0) {
        return [
            'price' => $storeSku['member_price'],
            'type' => 'member',
            'label' => '会员价'
        ];
    }
    
    // 优先级1：原价
    return [
        'price' => $storeSku['original_price'],
        'type' => 'original',
        'label' => ''
    ];
}

private function isOffpeakTime($serviceTime)
{
    if (!$serviceTime) {
        return false;
    }
    
    $hour = date('H', $serviceTime);
    // 闲时时段：14:00-17:00
    return ($hour >= 14 && $hour < 17);
}
```

### 5.2 应用优惠券

```php
public function applyC oupon($basePrice, $couponId, $userId)
{
    // 1. 获取优惠券信息
    $coupon = Db::name('coupon_user')
        ->alias('cu')
        ->join('coupon c', 'cu.coupon_id = c.id')
        ->where('cu.id', $couponId)
        ->where('cu.uid', $userId)
        ->where('cu.status', 0)
        ->find();
    
    if (!$coupon) {
        throw new Exception('优惠券不可用');
    }
    
    // 2. 检查使用门槛
    if ($basePrice < $coupon['use_min_price']) {
        throw new Exception('订单金额不满足使用条件');
    }
    
    // 3. 计算优惠金额
    $discountAmount = 0;
    
    switch ($coupon['coupon_type']) {
        case 1:  // 满减券
            $discountAmount = $coupon['coupon_price'];
            break;
            
        case 2:  // 折扣券
            $discountAmount = $basePrice * (1 - $coupon['coupon_price'] / 10);
            break;
            
        case 3:  // 随机金额券
            $discountAmount = $coupon['coupon_price'];
            break;
    }
    
    // 4. 计算最终价格
    $finalPrice = max($basePrice - $discountAmount, 0.01);  // 最低0.01元
    
    return [
        'base_price' => $basePrice,
        'coupon_discount' => $discountAmount,
        'final_price' => $finalPrice
    ];
}
```

## 六、前端展示逻辑

### 6.1 价格展示规则

```javascript
// 前端价格展示逻辑

function displayPrice(priceInfo) {
    const {
        original_price,
        base_price,
        price_type,
        price_label,
        discount_amount,
        coupon_discount,
        final_price
    } = priceInfo;
    
    // 1. 划线价（原价）
    const strikePrice = original_price;
    
    // 2. 到手价（最终价格）
    const actualPrice = final_price || base_price;
    
    // 3. 优惠标签
    const labels = [];
    if (price_label) {
        labels.push(price_label);
    }
    if (coupon_discount > 0) {
        labels.push(`券减¥${coupon_discount}`);
    }
    
    // 4. 优惠金额
    const totalDiscount = discount_amount + (coupon_discount || 0);
    
    return {
        strikePrice,      // 划线价
        actualPrice,      // 到手价
        labels,           // 优惠标签数组
        totalDiscount,    // 总优惠金额
        saveMoney: `已省¥${totalDiscount.toFixed(2)}`
    };
}
```

## 八、价格冲突处理

### 8.1 价格叠加规则

| 场景 | 处理规则 |
|------|----------|
| 会员+VIP | 取VIP价（优先级高） |
| VIP+闲时 | 取闲时价（优先级高） |
| 闲时+活动 | 取活动价（优先级高） |
| 任意价格+优惠券 | 在基础价格上再减优惠券 |

### 8.2 特殊规则
