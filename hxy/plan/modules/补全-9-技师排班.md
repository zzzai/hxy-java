## 一、问题描述

原方案提到"技师排班表（eb_technician_schedule）"，但**未给出具体表结构**，导致：

   - 闲时优惠依赖时间槽定价
   - 缺少数据结构支撑

### 4.3 时间槽状态流转

```
1=可预约
    ↓ 用户下单
2=已锁定（订单未支付）
    ↓ 支付成功
2=已锁定（订单已支付）
    ↓ 到店核销
3=已完成
    
或

2=已锁定（订单未支付）
    ↓ 超时未支付
1=可预约（释放时间槽）
```

### 4.4 并发预约控制

**问题**：多个用户同时预约同一技师同一时间槽

**解决方案**：数据库行锁 + Redis分布式锁

```php
// 1. Redis分布式锁（防止高并发）
$lockKey = "schedule_lock:{$scheduleId}";
$lock = Redis::set($lockKey, 1, ['NX', 'EX' => 10]);

if (!$lock) {
    return json(['code' => 400, 'msg' => '该时间槽正在被预约，请稍后重试']);
}

// 2. 数据库行锁（防止数据不一致）
Db::startTrans();
try {
    $schedule = Db::name('technician_schedule')
        ->where('id', $scheduleId)
        ->lock(true)  // 行锁
        ->find();
    
    if ($schedule['status'] != 1) {
        throw new Exception('该时间槽已被预约');
    }
    
    // 3. 更新状态
    Db::name('technician_schedule')
        ->where('id', $scheduleId)
        ->update([
            'status' => 2,
            'order_id' => $orderId,
            'lock_time' => time()
        ]);
    
    Db::commit();
    Redis::del($lockKey);
    
} catch (Exception $e) {
    Db::rollback();
    Redis::del($lockKey);
    return json(['code' => 400, 'msg' => $e->getMessage()]);
}
```

## 五、代码实现示例

### 5.1 批量生成排班

```php
// app/services/technician/ScheduleService.php

public function generateSchedule($technicianId, $startDate, $days = 7)
{
    $technician = Db::name('technician')->where('id', $technicianId)->find();
    if (!$technician) {
        throw new Exception('技师不存在');
    }
    
    // 1. 获取技师工作日历
    $calendars = Db::name('technician_calendar')
        ->where('technician_id', $technicianId)
        ->where('work_date', '>=', $startDate)
        ->where('work_date', '<', date('Y-m-d', strtotime($startDate . " +{$days} days")))
        ->where('is_workday', 1)
        ->select();
    
    // 2. 获取技师可服务的SKU列表
    $skus = Db::name('store_service_sku')
        ->alias('sss')
        ->join('service_sku ss', 'sss.sku_id = ss.id')
        ->where('sss.store_id', $technician['store_id'])
        ->where('sss.status', 1)
        ->where('ss.duration', '>', 0)  // 只有服务类SKU
        ->field('ss.id, ss.duration, sss.price, sss.offpeak_price')
        ->select();
    
    // 3. 遍历日历，生成时间槽
    foreach ($calendars as $calendar) {
        foreach ($skus as $sku) {
            $this->generateTimeSlots(
                $technicianId,
                $technician['store_id'],
                $sku['id'],
                $calendar['work_date'],
                $calendar['work_start_time'],
                $calendar['work_end_time'],
                $calendar['rest_start_time'],
                $calendar['rest_end_time'],
                $sku['duration'],
                $sku['price'],
                $sku['offpeak_price']
            );
        }
    }
}

private function generateTimeSlots($technicianId, $storeId, $skuId, $workDate, 
    $workStart, $workEnd, $restStart, $restEnd, $duration, $price, $offpeakPrice)
{
    $currentTime = strtotime($workDate . ' ' . $workStart);
    $endTime = strtotime($workDate . ' ' . $workEnd);
    $restStartTime = strtotime($workDate . ' ' . $restStart);
    $restEndTime = strtotime($workDate . ' ' . $restEnd);
    
    // 闲时时间段（14:00-17:00）
    $offpeakStart = strtotime($workDate . ' 14:00');
    $offpeakEnd = strtotime($workDate . ' 17:00');
    
    $slots = [];
    
    while ($currentTime + $duration * 60 <= $endTime) {
        // 跳过休息时间
        if ($currentTime >= $restStartTime && $currentTime < $restEndTime) {
            $currentTime = $restEndTime;
            continue;
        }
        
        $slotEnd = $currentTime + $duration * 60;
        
        // 判断是否闲时
        $isOffpeak = ($currentTime >= $offpeakStart && $currentTime < $offpeakEnd) ? 1 : 0;
        $slotPrice = $isOffpeak ? $offpeakPrice : $price;
        
        $slots[] = [
            'technician_id' => $technicianId,
            'store_id' => $storeId,
            'service_sku_id' => $skuId,
            'work_date' => $workDate,
            'time_slot_start' => date('H:i:s', $currentTime),
            'time_slot_end' => date('H:i:s', $slotEnd),
            'status' => 1,
            'is_offpeak' => $isOffpeak,
            'offpeak_price' => $slotPrice,
            'create_time' => time(),
            'update_time' => time()
        ];
        
        // 按15分钟颗粒度递增
        $currentTime += 15 * 60;
    }
    
    // 批量插入（忽略重复）
    if (!empty($slots)) {
        Db::name('technician_schedule')->insertAll($slots, true);
    }
}
```

### 5.2 查询可预约时间槽

```php
// app/api/controller/service/ScheduleController.php

public function getAvailableSlots(Request $request)
{
    $storeId = $request->param('store_id');
    $skuId = $request->param('sku_id');
    $date = $request->param('date', date('Y-m-d'));
    
    // 1. 查询该SKU在指定日期的所有可预约时间槽
    $slots = Db::name('technician_schedule')
        ->alias('ts')
        ->join('technician t', 'ts.technician_id = t.id')
        ->where('ts.store_id', $storeId)
        ->where('ts.service_sku_id', $skuId)
        ->where('ts.work_date', $date)
        ->where('ts.status', 1)  // 只查可预约的
        ->field('ts.*, t.name as technician_name, t.level as technician_level')
        ->order('ts.time_slot_start', 'asc')
        ->select();
    
    // 2. 按技师分组
    $result = [];
    foreach ($slots as $slot) {
        $technicianId = $slot['technician_id'];
        if (!isset($result[$technicianId])) {
            $result[$technicianId] = [
                'technician_id' => $technicianId,
                'technician_name' => $slot['technician_name'],
                'technician_level' => $slot['technician_level'],
                'slots' => []
            ];
        }
        
        $result[$technicianId]['slots'][] = [
            'id' => $slot['id'],
            'time' => substr($slot['time_slot_start'], 0, 5) . '-' . substr($slot['time_slot_end'], 0, 5),
            'is_offpeak' => $slot['is_offpeak'],
            'price' => $slot['is_offpeak'] ? $slot['offpeak_price'] : 0
        ];
    }
    
    return json([
        'code' => 200,
        'data' => array_values($result)
    ]);
}
```
