## 一、问题描述

原方案提到"套餐关联表"，但**未说明套餐定价逻辑**，导致：

### 业务痛点

1. **套餐价格不清**
   - 套餐 = 子项价格之和？还是有折扣？
   - 门店能否自定义套餐价格？

2. **折扣计算混乱**
   - 套餐打8折，是总价8折还是每个子项8折？
   - 子项价格变了，套餐价格要不要跟着变？

3. **可选子项定价不明**
   - 套餐包含可选子项，加项怎么收费？
   - 用户选了加项，总价怎么算？

4. **会员折扣叠加问题**
   - 套餐已经打折，会员还能再享受折扣吗？
   - 套餐价¥298，会员9折 = ¥268.20？

## 三、套餐定价模型

### 3.1 套餐价格构成

```
套餐总价 = 套餐基础价 + 可选子项加价 - 套餐折扣 - 会员折扣
```

### 3.2 定价类型

| 定价类型 | 说明 | 计算公式 | 示例 |
|---------|------|---------|------|
| **固定价** | 套餐固定价格，不随子项变动 | 套餐价 = 固定价 | ¥298 |
| **子项总价** | 套餐价 = 所有子项价格之和 | 套餐价 = Σ子项价 | ¥198+¥50+¥50=¥298 |
| **折扣价** | 在子项总价基础上打折 | 套餐价 = Σ子项价 × 折扣率 | ¥298 × 0.8 = ¥238.40 |
| **立减价** | 在子项总价基础上减固定金额 | 套餐价 = Σ子项价 - 立减额 | ¥298 - ¥50 = ¥248 |

## 五、套餐定价计算逻辑

### 5.1 计算套餐基础价

```php
// app/services/combo/ComboPriceService.php

public function calculateComboBasePrice($comboId, $storeId, $selectedItems = [])
{
    // 1. 获取套餐信息
    $combo = Db::name('service')
        ->where('id', $comboId)
        ->where('type', 3)  // 套餐类型
        ->find();
    
    // 2. 获取门店套餐信息
    $storeCombo = Db::name('store_service')
        ->where('service_id', $comboId)
        ->where('store_id', $storeId)
        ->find();
    
    // 3. 如果门店自定义了套餐价，直接返回
    if ($storeCombo['is_custom_combo_price'] == 1) {
        return [
            'base_price' => $storeCombo['custom_combo_price'],
            'price_type' => 'custom',
            'items_total' => 0,
            'discount' => 0
        ];
    }
    
    // 4. 获取套餐子项
    $items = Db::name('service_combo_relation')
        ->where('combo_service_id', $comboId)
        ->select();
    
    // 5. 计算子项总价
    $itemsTotal = 0;
    foreach ($items as $item) {
        // 必选子项
        if ($item['is_mandatory'] == 1) {
            $itemPrice = $this->getItemPrice($item['item_id'], $storeId);
            $itemsTotal += $itemPrice * $item['item_num'];
        }
        // 可选子项（用户选择的）
        elseif (in_array($item['id'], $selectedItems)) {
            $itemsTotal += $item['add_price'];
        }
    }
    
    // 6. 根据定价类型计算套餐价
    switch ($combo['combo_price_type']) {
        case 1:  // 固定价
            $basePrice = $combo['combo_base_price'];
            $discount = $itemsTotal - $basePrice;
            break;
            
        case 2:  // 子项总价
            $basePrice = $itemsTotal;
            $discount = 0;
            break;
            
        case 3:  // 折扣价
            $basePrice = $itemsTotal * $combo['combo_discount_rate'];
            $discount = $itemsTotal - $basePrice;
            break;
            
        case 4:  // 立减价
            $basePrice = $itemsTotal - $combo['combo_discount_amount'];
            $discount = $combo['combo_discount_amount'];
            break;
            
        default:
            $basePrice = $itemsTotal;
            $discount = 0;
    }
    
    return [
        'base_price' => $basePrice,
        'price_type' => $combo['combo_price_type'],
        'items_total' => $itemsTotal,
        'discount' => $discount,
        'save_amount' => $discount
    ];
}

private function getItemPrice($itemId, $storeId)
{
    $item = Db::name('store_service_sku')
        ->where('sku_id', $itemId)
        ->where('store_id', $storeId)
        ->find();
    
    return $item['price'] ?? 0;
}
```

### 5.2 应用会员折扣

```php
public function applyMemberDiscount($comboPrice, $userId, $comboId)
{
    // 1. 获取套餐信息
    $combo = Db::name('service')->where('id', $comboId)->find();
    
    // 2. 检查是否允许会员折扣
    if ($combo['allow_member_discount'] == 0) {
        return [
            'final_price' => $comboPrice,
            'member_discount' => 0,
            'reason' => '套餐价不可叠加会员折扣'
        ];
    }
    
    // 3. 获取用户会员信息
    $userLevel = Db::name('user')
        ->where('id', $userId)
        ->find();
    
    if (!$userLevel || $userLevel['level_type'] == 0) {
        return [
            'final_price' => $comboPrice,
            'member_discount' => 0,
            'reason' => '非会员用户'
        ];
    }
    
    // 4. 计算会员折扣
    $discountRate = $userLevel['level_discount_rate'] ?? 0.90;
    $memberDiscount = $comboPrice * (1 - $discountRate);
    $finalPrice = $comboPrice - $memberDiscount;
    
    return [
        'final_price' => $finalPrice,
        'member_discount' => $memberDiscount,
        'discount_rate' => $discountRate,
        'reason' => '会员' . ($discountRate * 10) . '折'
    ];
}
```

## 七、套餐价格展示规则

### 7.1 前端展示逻辑

```javascript
// 前端套餐价格展示

function displayComboPrice(comboInfo, userLevel) {
    const {
        items_total,      // 子项总价
        base_price,       // 套餐基础价
        price_type,       // 定价类型
        discount,         // 套餐折扣
        allow_member_discount  // 是否允许会员折扣
    } = comboInfo;
    
    // 1. 划线价（子项总价）
    const strikePrice = items_total;
    
    // 2. 套餐价
    let comboPrice = base_price;
    
    // 3. 会员折扣
    let memberDiscount = 0;
    if (allow_member_discount && userLevel) {
        memberDiscount = comboPrice * (1 - userLevel.discount_rate);
        comboPrice = comboPrice - memberDiscount;
    }
    
    // 4. 总优惠
    const totalSave = strikePrice - comboPrice;
    
    return {
        strikePrice: `¥${strikePrice.toFixed(2)}`,
        comboPrice: `¥${comboPrice.toFixed(2)}`,
        comboDiscount: `套餐立省¥${discount.toFixed(2)}`,
        memberDiscount: memberDiscount > 0 ? `会员再省¥${memberDiscount.toFixed(2)}` : '',
        totalSave: `共省¥${totalSave.toFixed(2)}`
    };
}
```

## 八、套餐价格变更处理

### 8.1 子项价格变更

```php
// 当子项价格变更时，套餐价格如何处理

public function handleItemPriceChange($itemId, $newPrice)
{
    // 1. 查询包含该子项的所有套餐
    $combos = Db::name('service_combo_relation')
        ->where('item_id', $itemId)
        ->column('combo_service_id');
    
    foreach ($combos as $comboId) {
        $combo = Db::name('service')->where('id', $comboId)->find();
        
        // 2. 根据定价类型处理
        switch ($combo['combo_price_type']) {
            case 1:  // 固定价 - 不变
                // 套餐价不变，但优惠金额会变
                break;
                
            case 2:  // 子项总价 - 自动变
                // 套餐价自动跟随子项价格变化
                $this->recalculateComboPrice($comboId);
                break;
                
            case 3:  // 折扣价 - 自动变
                // 套餐价 = 新的子项总价 × 折扣率
                $this->recalculateComboPrice($comboId);
                break;
                
            case 4:  // 立减价 - 自动变
                // 套餐价 = 新的子项总价 - 立减额
                $this->recalculateComboPrice($comboId);
                break;
        }
    }
}
```
