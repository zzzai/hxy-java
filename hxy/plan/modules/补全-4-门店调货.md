# 补全模块4：门店间调货逻辑

**优先级**：P1（重要）  
**影响范围**：库存管理、门店协作、用户体验  
**开发工作量**：1.5天

---

## 一、问题描述

原方案提到"实物商品"，但**未说明门店间调货逻辑**，导致：

### 业务痛点

1. **库存分布不均**
   - A店某商品库存100件，B店缺货
   - 用户在B店无法购买，体验差

2. **库存周转率低**
   - A店积压库存，B店频繁缺货
   - 整体库存利用率低

3. **调货流程不清**
   - 调货由谁发起？A店还是B店？
   - 调货成本谁承担？物流费、损耗？
   - 调货时间怎么承诺？用户等得了吗？

4. **财务结算困难**
   - 调货后，库存归属权如何变更？
   - 调货成本如何结算？

---

## 二、解决方案

### 方案核心：需求驱动 + 自动匹配 + 成本分摊 + 物流跟踪

---

## 三、调货业务模型

### 3.1 调货触发场景

| 场景 | 触发方式 | 处理流程 |
|------|----------|----------|
| **用户下单缺货** | 用户在B店下单，B店库存不足 | 系统自动查询附近门店库存，推荐调货 |
| **门店主动调货** | B店预判某商品即将缺货 | B店主动发起调货申请 |
| **总部调拨** | 总部统一调配库存 | 总部发起调拨指令 |

### 3.2 调货类型

| 类型 | 说明 | 成本承担 | 审批流程 |
|------|------|----------|----------|
| **紧急调货** | 用户已下单，需立即调货 | 调入方承担 | 自动审批 |
| **计划调货** | 门店预判需求，提前调货 | 调入方承担 | 需审批 |
| **总部调拨** | 总部统一调配 | 总部承担 | 无需审批 |

## 五、业务流程设计

### 5.1 紧急调货流程（用户下单触发）

```
用户在B店下单
    ↓
B店库存不足
    ↓
系统查询附近门店库存
    ↓
找到A店有库存
    ↓
提示用户：可从A店调货，预计X小时到货
    ↓
用户确认
    ↓
自动创建调货单（status=1，已审核）
    ↓
A店配货（status=2）
    ↓
A店发货（status=3）
    ↓
B店收货（status=4）
    ↓
B店入库（status=5）
    ↓
通知用户到店取货
    ↓
自动结算
```

### 5.2 计划调货流程（门店主动发起）

```
B店预判缺货
    ↓
B店发起调货申请（status=0）
    ↓
总部审核
    ↓ 通过
status=1（已审核）
    ↓
A店配货（status=2）
    ↓
... 后续流程同上
    
或

总部审核
    ↓ 拒绝
status=-1（已拒绝）
    ↓
通知B店
```

## 六、代码实现示例

### 6.1 查询附近门店库存

```php
// app/services/transfer/TransferService.php

public function findNearbyStock($skuId, $currentStoreId, $quantity)
{
    // 1. 获取当前门店坐标
    $currentStore = Db::name('store')->where('id', $currentStoreId)->find();
    
    // 2. 查询附近门店库存（50公里内）
    $nearbyStores = Db::name('store')
        ->alias('s')
        ->join('store_product_sku sps', 's.id = sps.store_id')
        ->where('sps.sku_id', $skuId)
        ->where('sps.available_stock', '>=', $quantity)
        ->where('s.id', '<>', $currentStoreId)
        ->where('s.status', 1)
        ->field('s.*, sps.available_stock, sps.price')
        ->select();
    
    // 3. 计算距离并排序
    $result = [];
    foreach ($nearbyStores as $store) {
        $distance = $this->calculateDistance(
            $currentStore['latitude'],
            $currentStore['longitude'],
            $store['latitude'],
            $store['longitude']
        );
        
        if ($distance <= 50) {  // 50公里内
            $store['distance'] = $distance;
            $store['estimated_time'] = $this->estimateDeliveryTime($distance);
            $result[] = $store;
        }
    }
    
    // 按距离排序
    usort($result, function($a, $b) {
        return $a['distance'] <=> $b['distance'];
    });
    
    return $result;
}

private function calculateDistance($lat1, $lng1, $lat2, $lng2)
{
    // 使用Haversine公式计算两点距离（公里）
    $earthRadius = 6371;
    
    $dLat = deg2rad($lat2 - $lat1);
    $dLng = deg2rad($lng2 - $lng1);
    
    $a = sin($dLat/2) * sin($dLat/2) +
         cos(deg2rad($lat1)) * cos(deg2rad($lat2)) *
         sin($dLng/2) * sin($dLng/2);
    
    $c = 2 * atan2(sqrt($a), sqrt(1-$a));
    
    return $earthRadius * $c;
}

private function estimateDeliveryTime($distance)
{
    // 简单估算：10公里内2小时，每增加10公里+1小时
    return ceil($distance / 10) * 2;
}
```

### 6.2 创建调货单

```php
public function createTransfer($fromStoreId, $toStoreId, $skuId, $quantity, 
    $transferType = 1, $orderId = 0)
{
    // 1. 获取SKU信息
    $sku = Db::name('service_sku')->where('id', $skuId)->find();
    $storeSku = Db::name('store_product_sku')
        ->where('store_id', $fromStoreId)
        ->where('sku_id', $skuId)
        ->find();
    
    // 2. 检查库存
    if ($storeSku['available_stock'] < $quantity) {
        throw new Exception('调出门店库存不足');
    }
    
    // 3. 生成调货单号
    $transferNo = 'TF' . date('YmdHis') . rand(1000, 9999);
    
    // 4. 计算金额
    $transferPrice = $storeSku['cost_price'] ?? $storeSku['price'] * 0.7;  // 成本价
    $transferAmount = $transferPrice * $quantity;
    $logisticsFee = $this->calculateLogisticsFee($fromStoreId, $toStoreId, $quantity);
    
    // 5. 确定成本承担方
    $costBearer = ($transferType == 3) ? 3 : 2;  // 总部调拨=总部承担，其他=调入方承担
    
    // 6. 创建调货单
    $transferId = Db::name('store_transfer')->insertGetId([
        'transfer_no' => $transferNo,
        'transfer_type' => $transferType,
        'from_store_id' => $fromStoreId,
        'to_store_id' => $toStoreId,
        'sku_id' => $skuId,
        'sku_name' => $sku['sku'],
        'transfer_quantity' => $quantity,
        'transfer_price' => $transferPrice,
        'transfer_amount' => $transferAmount,
        'logistics_fee' => $logisticsFee,
        'cost_bearer' => $costBearer,
        'order_id' => $orderId,
        'status' => ($transferType == 1) ? 1 : 0,  // 紧急调货自动审核
        'apply_user_id' => request()->uid ?? 0,
        'apply_time' => time()
    ]);
    
    // 7. 如果是紧急调货，立即锁定库存
    if ($transferType == 1) {
        $this->lockTransferStock($fromStoreId, $skuId, $quantity, $transferId);
    }
    
    return $transferId;
}

private function lockTransferStock($storeId, $skuId, $quantity, $transferId)
{
    // 锁定调出方库存
    Db::name('store_product_sku')
        ->where('store_id', $storeId)
        ->where('sku_id', $skuId)
        ->update([
            'available_stock' => Db::raw('available_stock - ' . $quantity),
            'locked_stock' => Db::raw('locked_stock + ' . $quantity),
            'update_time' => time()
        ]);
    
    // 记录库存流水
    Db::name('stock_flow')->insert([
        'store_id' => $storeId,
        'sku_id' => $skuId,
        'sku_type' => 2,
        'change_type' => 6,  // 调货锁定
        'change_quantity' => -$quantity,
        'operator_type' => 3,
        'remark' => '调货单锁定库存：' . $transferId,
        'create_time' => time()
    ]);
}
```

### 6.3 调货入库

```php
public function receiveTransfer($transferId)
{
    // 1. 获取调货单
    $transfer = Db::name('store_transfer')->where('id', $transferId)->find();
    
    if ($transfer['status'] != 4) {
        throw new Exception('调货单状态错误');
    }
    
    Db::startTrans();
    try {
        // 2. 调出方：扣减锁定库存，增加已售库存
        Db::name('store_product_sku')
            ->where('store_id', $transfer['from_store_id'])
            ->where('sku_id', $transfer['sku_id'])
            ->update([
                'locked_stock' => Db::raw('locked_stock - ' . $transfer['transfer_quantity']),
                'sold_stock' => Db::raw('sold_stock + ' . $transfer['transfer_quantity']),
                'update_time' => time()
            ]);
        
        // 3. 调入方：增加可售库存
        Db::name('store_product_sku')
            ->where('store_id', $transfer['to_store_id'])
            ->where('sku_id', $transfer['sku_id'])
            ->update([
                'available_stock' => Db::raw('available_stock + ' . $transfer['transfer_quantity']),
                'total_stock' => Db::raw('total_stock + ' . $transfer['transfer_quantity']),
                'update_time' => time()
            ]);
        
        // 4. 更新调货单状态
        Db::name('store_transfer')->where('id', $transferId)->update([
            'status' => 5,
            'receive_time' => time()
        ]);
        
        // 5. 创建结算记录
        $this->createSettlement($transfer);
        
        // 6. 如果关联订单，通知用户
        if ($transfer['order_id'] > 0) {
            $this->notifyUser($transfer['order_id']);
        }
        
        Db::commit();
        
    } catch (Exception $e) {
        Db::rollback();
        throw $e;
    }
}

private function createSettlement($transfer)
{
    $goodsAmount = $transfer['transfer_amount'];
    $logisticsFee = $transfer['logistics_fee'];
    $totalAmount = $goodsAmount + $logisticsFee;
    
    // 根据成本承担方计算
    switch ($transfer['cost_bearer']) {
        case 1:  // 调出方承担
            $fromStoreIncome = $goodsAmount;
            $toStoreCost = 0;
            $hqSubsidy = $logisticsFee;
            break;
            
        case 2:  // 调入方承担
            $fromStoreIncome = $goodsAmount;
            $toStoreCost = $totalAmount;
            $hqSubsidy = 0;
            break;
            
        case 3:  // 总部承担
            $fromStoreIncome = $goodsAmount;
            $toStoreCost = 0;
            $hqSubsidy = $totalAmount;
            break;
    }
    
    Db::name('transfer_settlement')->insert([
        'transfer_id' => $transfer['id'],
        'from_store_id' => $transfer['from_store_id'],
        'to_store_id' => $transfer['to_store_id'],
        'goods_amount' => $goodsAmount,
        'logistics_fee' => $logisticsFee,
        'total_amount' => $totalAmount,
        'from_store_income' => $fromStoreIncome,
        'to_store_cost' => $toStoreCost,
        'hq_subsidy' => $hqSubsidy,
        'settlement_status' => 0,
        'create_time' => time()
    ]);
}
```
