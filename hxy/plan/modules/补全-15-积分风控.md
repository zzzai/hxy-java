# 补全模块15：积分兑换风控设计

**优先级**：P2（建议）  
**影响范围**：积分系统、风控保护、营销成本  
**开发工作量**：1天

---

## 一、问题描述

原方案提到"积分兑换SKU"，但**未考虑风控**，导致：

### 业务痛点

1. **积分刷单**
   - 恶意用户通过刷单获取大量积分
   - 批量兑换商品后转卖

2. **积分无限累积**
   - 积分没有有效期
   - 用户囤积大量积分

3. **兑换无限制**
   - 单次兑换数量不限
   - 每日兑换次数不限

4. **黑产攻击**
   - 批量注册账号刷积分
   - 薅羊毛行为

---

## 二、解决方案

### 方案核心：限额控制 + 有效期管理 + 审核机制 + 黑名单

### 5.1 兑换前检查

```php
// app/services/points/PointsExchangeService.php

public function checkExchangeLimit($userId, $skuId, $pointsCost, $quantity)
{
    // 1. 检查实名认证
    $user = Db::name('user')->where('id', $userId)->find();
    
    if (!$user['id_card']) {
        throw new Exception('积分兑换需要实名认证');
    }
    
    // 2. 检查注册时间
    $registerDays = (time() - $user['create_time']) / 86400;
    
    if ($registerDays < 7) {
        throw new Exception('注册7天后才能兑换');
    }
    
    // 3. 检查黑名单
    $blacklist = Db::name('points_blacklist')
        ->where('user_id', $userId)
        ->where('blacklist_type', 'in', [2, 3])  // 禁止兑换或全部禁止
        ->where(function($query) {
            $query->where('end_time', 0)
                  ->whereOr('end_time', '>', time());
        })
        ->find();
    
    if ($blacklist) {
        throw new Exception('您已被限制兑换积分，原因：' . $blacklist['reason']);
    }
    
    // 4. 检查可用积分
    $userPoints = Db::name('user_points')->where('user_id', $userId)->find();
    
    if ($userPoints['available_points'] < $pointsCost * $quantity) {
        throw new Exception('积分不足');
    }
    
    // 5. 检查单次兑换数量
    if ($quantity > 1) {
        throw new Exception('单次最多兑换1件商品');
    }
    
    // 6. 检查每日兑换次数
    $todayCount = Db::name('points_exchange_record')
        ->where('user_id', $userId)
        ->where('create_time', '>=', strtotime('today'))
        ->count();
    
    if ($todayCount >= 3) {
        throw new Exception('今日兑换次数已达上限');
    }
    
    // 7. 检查每月兑换次数
    $monthCount = Db::name('points_exchange_record')
        ->where('user_id', $userId)
        ->where('create_time', '>=', strtotime('first day of this month'))
        ->count();
    
    if ($monthCount >= 10) {
        throw new Exception('本月兑换次数已达上限');
    }
    
    // 8. 检查是否需要审核
    $needAudit = ($pointsCost * $quantity >= 1000);
    
    return [
        'can_exchange' => true,
        'need_audit' => $needAudit
    ];
}
```

### 5.2 积分兑换

```php
public function exchange($userId, $skuId, $quantity = 1)
{
    // 1. 获取兑换规则
    $exchangeRule = Db::name('store_points_exchange')
        ->where('sku_id', $skuId)
        ->where('status', 1)
        ->find();
    
    if (!$exchangeRule) {
        throw new Exception('该商品不可兑换');
    }
    
    $pointsCost = $exchangeRule['points_cost'];
    
    // 2. 检查限制
    $check = $this->checkExchangeLimit($userId, $skuId, $pointsCost, $quantity);
    
    Db::startTrans();
    try {
        // 3. 扣减积分
        $this->deductPoints($userId, $pointsCost * $quantity, 'exchange', 0);
        
        // 4. 创建兑换记录
        $status = $check['need_audit'] ? 0 : 1;  // 需要审核则待审核，否则直接通过
        
        $recordId = Db::name('points_exchange_record')->insertGetId([
            'user_id' => $userId,
            'sku_id' => $skuId,
            'sku_name' => $exchangeRule['sku_name'],
            'points_cost' => $pointsCost * $quantity,
            'quantity' => $quantity,
            'status' => $status,
            'create_time' => time()
        ]);
        
        // 5. 如果不需要审核，直接创建订单
        if (!$check['need_audit']) {
            $this->createExchangeOrder($recordId);
        }
        
        Db::commit();
        
        return [
            'record_id' => $recordId,
            'need_audit' => $check['need_audit'],
            'message' => $check['need_audit'] ? '兑换申请已提交，等待审核' : '兑换成功'
        ];
        
    } catch (Exception $e) {
        Db::rollback();
        throw $e;
    }
}

private function deductPoints($userId, $points, $sourceType, $sourceId)
{
    // 扣减可用积分
    Db::name('user_points')
        ->where('user_id', $userId)
        ->update([
            'available_points' => Db::raw('available_points - ' . $points),
            'used_points' => Db::raw('used_points + ' . $points)
        ]);
    
    // 记录流水
    $userPoints = Db::name('user_points')->where('user_id', $userId)->find();
    
    Db::name('points_flow')->insert([
        'user_id' => $userId,
        'change_type' => 2,  // 使用
        'change_amount' => -$points,
        'before_points' => $userPoints['available_points'] + $points,
        'after_points' => $userPoints['available_points'],
        'source_type' => $sourceType,
        'source_id' => $sourceId,
        'remark' => '积分兑换',
        'create_time' => time()
    ]);
}
```

### 5.3 大额兑换审核

```php
public function auditExchange($recordId, $status, $remark = '')
{
    $record = Db::name('points_exchange_record')->where('id', $recordId)->find();
    
    if ($record['status'] != 0) {
        throw new Exception('该记录不是待审核状态');
    }
    
    Db::startTrans();
    try {
        if ($status == 1) {
            // 审核通过
            Db::name('points_exchange_record')
                ->where('id', $recordId)
                ->update([
                    'status' => 1,
                    'audit_user_id' => request()->uid ?? 0,
                    'audit_time' => time(),
                    'audit_remark' => $remark
                ]);
            
            // 创建订单
            $this->createExchangeOrder($recordId);
            
        } else {
            // 审核拒绝，退还积分
            Db::name('points_exchange_record')
                ->where('id', $recordId)
                ->update([
                    'status' => 2,
                    'audit_user_id' => request()->uid ?? 0,
                    'audit_time' => time(),
                    'audit_remark' => $remark
                ]);
            
            // 退还积分
            $this->refundPoints($record['user_id'], $record['points_cost'], 'exchange_reject', $recordId);
        }
        
        Db::commit();
        
    } catch (Exception $e) {
        Db::rollback();
        throw $e;
    }
}
```

```php
// app/command/ExpirePoints.php

protected function execute(Input $input, Output $output)
{
    // 每天凌晨执行
    
    // 查询已过期的积分
    $expiredFlows = Db::name('points_flow')
        ->where('change_type', 1)  // 获得积分
        ->where('expire_time', '>', 0)
        ->where('expire_time', '<', time())
        ->where('is_expired', 0)
        ->select();
    
    foreach ($expiredFlows as $flow) {
        // 计算该笔积分剩余多少未使用
        $remainingPoints = $this->calculateRemainingPoints($flow);
        
        if ($remainingPoints > 0) {
            // 扣减可用积分
            Db::name('user_points')
                ->where('user_id', $flow['user_id'])
                ->update([
                    'available_points' => Db::raw('available_points - ' . $remainingPoints),
                    'expired_points' => Db::raw('expired_points + ' . $remainingPoints)
                ]);
            
            // 记录过期流水
            Db::name('points_flow')->insert([
                'user_id' => $flow['user_id'],
                'change_type' => 3,  // 过期
                'change_amount' => -$remainingPoints,
                'source_type' => 'expire',
                'source_id' => $flow['id'],
                'remark' => '积分过期',
                'create_time' => time()
            ]);
            
            $output->writeln('用户' . $flow['user_id'] . '过期积分：' . $remainingPoints);
        }
        
        // 标记为已过期
        Db::name('points_flow')->where('id', $flow['id'])->update(['is_expired' => 1]);
    }
}
```

## 六、异常行为检测

```php
// app/services/risk/PointsRiskService.php

public function detectAbnormalBehavior($userId)
{
    $risks = [];
    
    // 1. 检测短时间大量获得积分
    $recentPoints = Db::name('points_flow')
        ->where('user_id', $userId)
        ->where('change_type', 1)
        ->where('create_time', '>=', time() - 3600)
        ->sum('change_amount');
    
    if ($recentPoints > 1000) {
        $risks[] = '1小时内获得' . $recentPoints . '积分，疑似刷单';
    }
    
    // 2. 检测频繁兑换
    $todayExchangeCount = Db::name('points_exchange_record')
        ->where('user_id', $userId)
        ->where('create_time', '>=', strtotime('today'))
        ->count();
    
    if ($todayExchangeCount >= 3) {
        $risks[] = '今日兑换' . $todayExchangeCount . '次，达到上限';
    }
    
    // 3. 检测设备关联
    $user = Db::name('user')->where('id', $userId)->find();
    $sameDeviceUsers = Db::name('user')
        ->where('device_id', $user['device_id'])
        ->where('id', '<>', $userId)
        ->count();
    
    if ($sameDeviceUsers >= 5) {
        $risks[] = '同设备' . ($sameDeviceUsers + 1) . '个账号，疑似批量注册';
    }
    
    // 4. 如果有风险，加入黑名单
    if (!empty($risks)) {
        $this->addToBlacklist($userId, implode('；', $risks));
    }
    
    return $risks;
}

private function addToBlacklist($userId, $reason)
{
    Db::name('points_blacklist')->insert([
        'user_id' => $userId,
        'blacklist_type' => 3,  // 全部禁止
        'reason' => $reason,
        'start_time' => time(),
        'end_time' => strtotime('+30 days'),
        'create_time' => time()
    ]);
}
```
