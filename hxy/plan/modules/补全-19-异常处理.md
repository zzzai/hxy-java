# 补全模块19：订单异常处理机制

**优先级**：P2（建议）  
**影响范围**：用户体验、客诉处理、业务连续性  
**开发工作量**：1天

---

## 一、问题描述

原方案缺少**订单异常场景的处理机制**，导致：

### 业务痛点

1. **技师临时请假**
   - 用户已预约，技师突然请假
   - 没有自动通知和改期机制

2. **实物库存突发短缺**
   - 订单已支付，发现库存不足
   - 没有补偿方案

3. **门店突发闭店**
   - 用户预约当天，门店临时闭店
   - 没有转单或退款机制

4. **套餐子项变更**
   - 总部修改套餐子项
   - 历史订单不知道如何处理

---

## 二、解决方案

### 方案核心：异常检测 + 自动通知 + 补偿方案 + 兜底机制

---

### 3.1 异常类型

| 异常类型 | 触发条件 | 处理优先级 | 影响范围 |
|---------|---------|-----------|---------|
| **技师请假** | 技师标记请假 | P0 | 该技师的所有预约 |
| **库存不足** | 实物库存<订单数量 | P1 | 该SKU的所有订单 |

| **价格异常** | 价格突然变动 | P2 | 待支付订单 |
| **支付超时** | 30分钟未支付 | P1 | 待支付订单 |

### 5.1 技师请假处理

```php
// app/services/exception/TechnicianLeaveService.php

public function handleTechnicianLeave($technicianId, $leaveDate, $reason)
{
    // 1. 查询该技师在请假日期的所有预约
    $schedules = Db::name('technician_schedule')
        ->where('technician_id', $technicianId)
        ->where('work_date', $leaveDate)
        ->where('status', 2)  // 已锁定（有预约）
        ->select();
    
    foreach ($schedules as $schedule) {
        // 2. 创建异常记录
        $exceptionId = Db::name('order_exception')->insertGetId([
            'order_id' => $schedule['order_id'],
            'exception_type' => 'technician_leave',
            'exception_reason' => '技师请假：' . $reason,
            'exception_detail' => json_encode([
                'technician_id' => $technicianId,
                'leave_date' => $leaveDate,
                'schedule_id' => $schedule['id']
            ]),
            'handle_status' => 0,
            'create_time' => time()
        ]);
        
        // 3. 查询替代技师
        $alternativeTechnicians = $this->findAlternativeTechnicians(
            $schedule['store_id'],
            $schedule['service_sku_id'],
            $schedule['work_date'],
            $schedule['time_slot_start']
        );
        
        // 4. 通知用户
        $this->notifyUser($schedule['order_id'], [
            'type' => 'technician_leave',
            'message' => '您预约的技师临时请假',
            'alternatives' => $alternativeTechnicians,
            'options' => [
                '1' => '更换其他技师',
                '2' => '改期',
                '3' => '退款'
            ]
        ]);
    }
    
    // 5. 更新排班状态
    Db::name('technician_schedule')
        ->where('technician_id', $technicianId)
        ->where('work_date', $leaveDate)
        ->update(['status' => 4]);  // 技师请假
}

private function findAlternativeTechnicians($storeId, $skuId, $date, $timeSlot)
{
    return Db::name('technician_schedule')
        ->alias('ts')
        ->join('technician t', 'ts.technician_id = t.id')
        ->where('ts.store_id', $storeId)
        ->where('ts.service_sku_id', $skuId)
        ->where('ts.work_date', $date)
        ->where('ts.time_slot_start', $timeSlot)
        ->where('ts.status', 1)  // 可预约
        ->field('t.id, t.name, t.level')
        ->select();
}
```

### 5.2 库存不足处理

```php
// app/services/exception/StockShortageService.php

public function handleStockShortage($orderId, $itemId, $requiredQuantity, $actualQuantity)
{
    // 1. 创建异常记录
    $exceptionId = Db::name('order_exception')->insertGetId([
        'order_id' => $orderId,
        'exception_type' => 'stock_shortage',
        'exception_reason' => '库存不足',
        'exception_detail' => json_encode([
            'item_id' => $itemId,
            'required' => $requiredQuantity,
            'actual' => $actualQuantity,
            'shortage' => $requiredQuantity - $actualQuantity
        ]),
        'handle_status' => 0,
        'create_time' => time()
    ]);
    
    // 2. 查询附近门店库存
    $order = Db::name('order')->where('id', $orderId)->find();
    $nearbyStores = $this->findNearbyStoresWithStock(
        $order['store_id'],
        $itemId,
        $requiredQuantity
    );
    
    // 3. 自动处理方案
    if (!empty($nearbyStores)) {
        // 方案A：从附近门店调货
        $this->proposeTransfer($exceptionId, $nearbyStores[0]);
    } else {
        // 方案B：补偿方案
        $this->proposeCompensation($exceptionId, $order['user_id'], $order['pay_price']);
    }
    
    // 4. 通知用户
    $this->notifyUser($orderId, [
        'type' => 'stock_shortage',
        'message' => '商品库存不足',
        'solutions' => [
            '1' => '从附近门店调货（预计X小时到货）',
            '2' => '更换同类商品',
            '3' => '退款并获得补偿'
        ]
    ]);
}

private function proposeCompensation($exceptionId, $userId, $orderAmount)
{
    // 补偿方案：退款 + 20%优惠券
    $compensationAmount = $orderAmount * 0.2;
    
    Db::name('order_compensation')->insert([
        'order_id' => $orderId,
        'user_id' => $userId,
        'compensation_type' => 1,  // 优惠券
        'compensation_amount' => $compensationAmount,
        'compensation_reason' => '库存不足补偿',
        'status' => 0,
        'create_time' => time()
    ]);
    
    // 更新异常处理方案
    Db::name('order_exception')
        ->where('id', $exceptionId)
        ->update([
            'handle_solution' => 'compensate',
            'handle_result' => json_encode([
                'refund' => $orderAmount,
                'coupon' => $compensationAmount
            ])
        ]);
}
```

### 5.3 门店闭店处理

```php
// app/services/exception/StoreClosureService.php

public function handleStoreClosure($storeId, $closureDate, $reason)
{
    // 1. 查询该门店在闭店日期的所有预约
    $orders = Db::name('order')
        ->where('store_id', $storeId)
        ->where('service_date', $closureDate)
        ->where('status', 'in', [0, 1])  // 待支付或已支付
        ->select();
    
    foreach ($orders as $order) {
        // 2. 创建异常记录
        $exceptionId = Db::name('order_exception')->insertGetId([
            'order_id' => $order['id'],
            'exception_type' => 'store_closure',
            'exception_reason' => '门店临时闭店：' . $reason,
            'exception_detail' => json_encode([
                'store_id' => $storeId,
                'closure_date' => $closureDate
            ]),
            'handle_status' => 0,
            'create_time' => time()
        ]);
        
        // 3. 查询附近门店
        $nearbyStores = $this->findNearbyStores($storeId, 20);  // 20公里内
        
        if (!empty($nearbyStores)) {
            // 方案A：转单到附近门店
            $this->proposeTransferStore($exceptionId, $nearbyStores);
        } else {
            // 方案B：全额退款 + 补偿
            $this->proposeFullRefund($exceptionId, $order);
        }
        
        // 4. 通知用户
        $this->notifyUser($order['id'], [
            'type' => 'store_closure',
            'message' => '门店临时闭店',
            'reason' => $reason,
            'solutions' => [
                '1' => '转到附近门店',
                '2' => '改期',
                '3' => '全额退款并获得补偿'
            ]
        ]);
    }
}
```

### 5.4 套餐变更处理

```php
// app/services/exception/ComboChangeService.php

public function handleComboChange($comboId, $oldVersion, $newVersion)
{
    // 1. 查询使用旧版本套餐的未完成订单
    $orders = Db::name('order')
        ->where('combo_id', $comboId)
        ->where('combo_version', $oldVersion)
        ->where('status', '<', 3)  // 未完成
        ->select();
    
    foreach ($orders as $order) {
        // 2. 创建异常记录
        Db::name('order_exception')->insert([
            'order_id' => $order['id'],
            'exception_type' => 'combo_change',
            'exception_reason' => '套餐子项变更',
            'exception_detail' => json_encode([
                'combo_id' => $comboId,
                'old_version' => $oldVersion,
                'new_version' => $newVersion
            ]),
            'handle_status' => 2,  // 已处理（自动处理）
            'handle_solution' => 'keep_old_version',
            'handle_result' => json_encode([
                'message' => '历史订单保持旧版本套餐内容'
            ]),
            'create_time' => time()
        ]);
    }
    
    // 历史订单继续使用旧版本，无需用户操作
}
```

### 6.1 用户选择改期

```php
public function handleReschedule($exceptionId, $newDate, $newTimeSlot)
{
    $exception = Db::name('order_exception')->where('id', $exceptionId)->find();
    $order = Db::name('order')->where('id', $exception['order_id'])->find();
    
    // 1. 释放原时间槽
    Db::name('technician_schedule')
        ->where('order_id', $order['id'])
        ->update(['status' => 1, 'order_id' => 0]);
    
    // 2. 锁定新时间槽
    Db::name('technician_schedule')
        ->where('id', $newTimeSlot)
        ->update(['status' => 2, 'order_id' => $order['id']]);
    
    // 3. 更新订单
    Db::name('order')
        ->where('id', $order['id'])
        ->update(['service_date' => $newDate]);
    
    // 4. 更新异常状态
    Db::name('order_exception')
        ->where('id', $exceptionId)
        ->update([
            'handle_status' => 2,
            'handle_solution' => 'reschedule',
            'handle_result' => json_encode([
                'new_date' => $newDate,
                'new_time_slot' => $newTimeSlot
            ]),
            'handle_time' => time()
        ]);
}
```

```php
// app/services/compensation/CompensationService.php

public function issueCompensation($compensationId)
{
    $compensation = Db::name('order_compensation')->where('id', $compensationId)->find();
    
    switch ($compensation['compensation_type']) {
        case 1:  // 优惠券
            $this->issueCoupon(
                $compensation['user_id'],
                $compensation['compensation_amount']
            );
            break;
            
        case 2:  // 积分
            $this->issuePoints(
                $compensation['user_id'],
                $compensation['compensation_amount']
            );
            break;
            
        case 3:  // 现金
            $this->issueRefund(
                $compensation['user_id'],
                $compensation['compensation_amount']
            );
            break;
            
        case 4:  // 免费服务
            $this->issueFreeService(
                $compensation['user_id'],
                $compensation['compensation_amount']
            );
            break;
    }
    
    // 更新补偿状态
    Db::name('order_compensation')
        ->where('id', $compensationId)
        ->update(['status' => 1, 'issue_time' => time()]);
}
```

## 十、总结

### 核心价值
1. **自动检测**：系统自动检测异常情况
2. **主动通知**：第一时间通知用户
3. **多种方案**：提供多种处理方案供用户选择
4. **自动补偿**：系统自动发放补偿

### 异常处理流程
```
异常发生 → 创建异常记录 → 查找解决方案 → 通知用户 → 用户选择 → 执行方案 → 发放补偿
```
