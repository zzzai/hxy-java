# 荷小悦小程序接口设计文档

> **基于CRMEB二次开发**  
> **协议**: RESTful API  
> **数据格式**: JSON  
> **版本**: V1.0

---

## 一、接口规范

### 1.1 基础信息

| 项目 | 说明 |
|------|------|
| **Base URL** | `https://api.hexiaoyue.com/api/` |
| **请求方式** | GET / POST |
| **编码格式** | UTF-8 |
| **Content-Type** | `application/json` |

### 1.2 通用响应格式

```json
{
  "status": 200,
  "msg": "success",
  "data": {}
}
```

### 1.3 状态码说明

| 状态码 | 说明 |
|--------|------|
| 200 | 请求成功 |
| 400 | 参数错误 |
| 401 | 未授权/Token过期 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |
| 1001 | 业务逻辑错误（自定义） |

### 1.4 认证方式

```
Header: Authorization: Bearer {token}
```
