## 📦 组件使用方法

### 1. 在页面中引入

```vue
<template>
  <view>
    <!-- 会员卡片组件 -->
    <member-card ref="memberCard" :autoLoad="true"></member-card>
  </view>
</template>

<script>
import MemberCard from '@/components/MemberCard/MemberCard.vue';

export default {
  components: {
    MemberCard
  },
  methods: {
    // 刷新会员数据
    refreshMember() {
      this.$refs.memberCard.refresh();
    }
  }
}
</script>
```

### 2. 组件属性

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| autoLoad | Boolean | true | 是否自动加载数据 |

### 3. 组件方法

| 方法名 | 说明 | 参数 |
|--------|------|------|
| refresh() | 刷新会员数据 | 无 |
| loadMemberData() | 加载会员数据 | 无 |
