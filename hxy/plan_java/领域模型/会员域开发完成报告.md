## 🎯 核心业务规则

### 会员等级体系

| 等级 | 名称 | 所需成长值 | 折扣 |
|------|------|-----------|------|
| 0 | 青铜会员 | 0 | 95% |
| 1 | 白银会员 | 1000 | 90% |
| 2 | 黄金会员 | 5000 | 85% |
| 3 | 铂金会员 | 10000 | 80% |
| 4 | 钻石会员 | 20000 | 75% |

### 成长值规则

- **获取方式**：消费1元 = 1成长值
- **升级规则**：成长值达到阈值自动升级
- **降级规则**：暂不支持降级

### 折扣计算

```java
折扣价 = 原价 × 会员等级折扣
```

Member member = Member.create(userId);

```java
Member member = memberRepository.findByUserId(userId);
member.addGrowthValue(new BigDecimal("100.00")); // 消费100元
memberRepository.save(member);
// 如果成长值达到阈值，会自动升级
```

```java
Member member = memberRepository.findByUserId(userId);
BigDecimal discountPrice = member.calculateDiscountPrice(new BigDecimal("128.00"));
```

## 🎯 领域事件流程

### 预约完成 → 增加成长值 → 自动升级

```
1. 用户完成预约服务
   ↓
2. Booking.complete() 发布 BookingCompletedEvent
   ↓
3. MemberEventListener 监听事件
   ↓
4. 查询会员（不存在则自动创建）
   ↓
5. member.addGrowthValue(payAmount)
   ↓
6. 成长值达到阈值 → 自动升级
   ↓
7. 发布 MemberUpgradedEvent（可选）
```

## 🎯 技术亮点

### 1. 充血模型

```java
// ✅ 业务规则在聚合根中
public class Member {
    public void addGrowthValue(BigDecimal amount) {
        this.growthValue = this.growthValue.add(amount.intValue());
        
        // 自动升级检查
        if (this.growthValue.canUpgrade(this.level)) {
            this.upgrade();
        }
    }
}
```

### 2. 值对象不可变

```java
public class GrowthValue {
    private final int value;  // final 不可变
    
    public GrowthValue add(int amount) {
        return new GrowthValue(this.value + amount);  // 返回新对象
    }
}
```

### 3. 领域事件解耦

```java
// 预约域 → 会员域（通过事件，无直接依赖）
@EventListener
public void onBookingCompleted(BookingCompletedEvent event) {
    member.addGrowthValue(event.getPayAmount());
}
```
