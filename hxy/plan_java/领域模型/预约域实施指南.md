## 🎯 核心业务规则

### 规则1：预约时间校验

```java
private static void validateBookingTime(Date bookingTime) {
    Date now = new Date();
    
    // 1. 不能预约过去的时间
    if (bookingTime.before(now)) {
        throw new IllegalArgumentException("不能预约过去的时间");
    }
    
    // 2. 至少提前30分钟预约
    long diff = bookingTime.getTime() - now.getTime();
    if (diff < 30 * 60 * 1000) {
        throw new IllegalArgumentException("请至少提前30分钟预约");
    }
    
    // 3. 不能预约30天后的时间
    if (diff > 30L * 24 * 60 * 60 * 1000) {
        throw new IllegalArgumentException("最多只能预约30天内的服务");
    }
    
    // 4. 检查营业时间（10:00-22:00）
    int hour = bookingTime.getHours();
    if (hour < 10 || hour >= 22) {
        throw new IllegalArgumentException("营业时间为10:00-22:00");
    }
}
```

### 规则2：状态流转控制

```
待确认(0) → 已确认(1) → 服务中(2) → 已完成(3)
    ↓
  已取消(4) / 超时取消(5)
```

```java
public void cancel(String reason) {
    if (!status.canCancel()) {
        throw new IllegalStateException("当前状态不能取消预约");
    }
    this.status = BookingStatus.CANCELLED;
    this.cancelReason = reason;
}
```

### 规则3：改期限制

```java
public boolean canReschedule() {
    if (status != BookingStatus.CONFIRMED) {
        return false;
    }
    // 距离预约时间2小时以上才能改期
    long diff = bookingTime.getTime() - new Date().getTime();
    return diff > 2 * 60 * 60 * 1000;
}
```

### BookingRepositoryImpl实现

**核心职责**：领域对象 ↔ 持久化对象转换

```java
@Repository
public class BookingRepositoryImpl implements BookingRepository {
    @Autowired
    private BookingDao bookingDao;
    
    public Booking save(Booking booking) {
        BookingPO po = toPO(booking);  // 领域对象 → PO
        bookingDao.insert(po);
        booking.setId(po.getId());
        return booking;
    }
    
    public Booking findById(Integer id) {
        BookingPO po = bookingDao.selectById(id);
        return toDomain(po);  // PO → 领域对象
    }
}
```
