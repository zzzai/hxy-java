# 荷小悦知识库文档导航（目录分类版）

> 目标：按“基线/实施/指南/数据/外部/归档”分层，降低查找成本。

## 1. 目录分类

| 目录 | 作用 | 是否主线 |
|------|------|----------|
| `baseline/` | 产品与架构基线文档 | 是 |
| `implementation/` | 分周实施计划与落地步骤 | 是 |
| `guides/` | API、前端、部署、执行指南 | 是 |
| `database/` | SQL基线与补丁脚本 | 是 |
| `reports/` | 状态与评审报告 | 辅助 |
| `external/structured/` | 外部资料结构化版本 | 是（外部输入） |
| `external/raw/` | 外部资料原始文件 | 否（追溯） |
| `plan/` | 模块补全与专项方案库 | 有条件采用 |
| `plan_java/` | DDD与演进方法论 | 中长期主线 |
| `archive/` | 历史过程文档 | 否（仅追溯） |
| `tools/` | 转换脚本与环境脚本 | 工具 |
